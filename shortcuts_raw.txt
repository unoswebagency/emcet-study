003: Chapter 1 - Matrices & Determinants 
004: GROUP 1: Determinant Value Hacks (Zero Calculation)
005: 1. The Arithmetic Progression (A.P.) Hack
006: Shortcut: If the elements of any row or column of a 3 × 3 determinant are in A.P., the determinant is ALWAYS 0.
007: Example: |2 4 6; 1 5 9; 10 20 30|. Look at Row 1 (common diff = 2) and Row 2 (common diff = 4). Answer = 0.
008: 2. The Logarithmic Geometric Progression (G.P.) Hack
009: Shortcut: If the elements of a determinant are log values of terms in a G.P., the determinant is ALWAYS 0.
010: Example: |log 2 log 4 log 8; log 3 log 9 log 27; log 5 log 25 log 125|. (Since 2, 4, 8 is a G.P., their logs form an A.P.). Answer = 0.
011: 3. Odd-Order Skew-Symmetric Hack
012: Shortcut: The determinant of ANY skew-symmetric matrix (Aᵀ = -A) of odd order (3 × 3, 5 × 5) is ALWAYS 0.
013: Example: |0 7 -4; -7 0 5; 4 -5 0|. (Diagonals are 0, and aᵢⱼ = -aⱼᵢ). Answer = 0.
014: 4. Even-Order Skew-Symmetric Hack
015: Shortcut: The determinant of a skew-symmetric matrix of even order (2 × 2, 4 × 4) is ALWAYS a Perfect Square.
016: Example: If asked for the determinant of a 4 × 4 skew-symmetric matrix and the options are (A) 14, (B) 16, (C) 18, (D) 20. Answer = 16 (only perfect square).
017: 5. Maximum Value with {0, 1}
018: Shortcut: The maximum possible value of a 3 × 3 determinant whose elements are strictly 0 or 1 is exactly 2.
019: Example: "What is the maximum value of a determinant formed using only the binary digits 0 and 1?" Answer = 2.
020: 6. Maximum Value with {-1, 1}
021: Shortcut: The maximum possible value of a 3 × 3 determinant whose elements are strictly -1 or 1 is exactly 4.
022: Example: "What is the max value of a 3 × 3 matrix A where aᵢⱼ ∈ {-1, 1}?" Answer = 4.
023: 7. Diagonal/Triangular Product Rule
024: Shortcut: For any upper triangular, lower triangular, or diagonal matrix, the determinant is simply the product of its principal diagonal elements.
025: Example: |5 8 9; 0 2 7; 0 0 -3|. Answer = 5 × 2 × (-3) = -30.
026: 8. The Vandermonde Standard
027: Shortcut: |1 1 1; a b c; a² b² c²| = (a-b)(b-c)(c-a).
028: Example: |1 1 1; 4 5 6; 16 25 36|. Here a=4, b=5, c=6. Answer = (4-5)(5-6)(6-4) = (-1)(-1)(2) = 2.
029: 9. The Circulant Root Hack
030: Shortcut: For |x a a; a x a; a a x| = 0, the roots are ALWAYS -2a, a, a.
031: Example: |x 5 5; 5 x 5; 5 5 x| = 0. Answer: Roots are -10, 5, 5.
032: 10. Cyclic Symmetric Expansion
033: Shortcut: |a b c; b c a; c a b| = -(a³ + b³ + c³ - 3abc).
034: Example: |1 2 3; 2 3 1; 3 1 2|. Answer = -(1³ + 2³ + 3³ - 3(1)(2)(3)) = -(1 + 8 + 27 - 18) = -18.
035: GROUP 2: Adjoint & Inverse Hacks
036: 11. The 2 times 2 Inverse Flash-Solve
037: Shortcut: For A = [a b; c d], A⁻¹ = 1/(ad-bc) [d -b; -c a]. (Swap main diagonal, flip signs of the other).
038: Example: A = [2 3; 1 4]. Answer: A⁻¹ = 1/5 [4 -3; -1 2].
039: 12. Adjoint Determinant Rule
040: Shortcut: |adj(A)| = |A|ⁿ⁻¹. For a 3 × 3 matrix, it is simply |A|².
041: Example: If A is a 3 × 3 matrix and |A| = 4, find |adj A|. Answer = 4² = 16.
042: 13. Double Adjoint Determinant Rule
043: Shortcut: |adj(adj(A))| = |A|⁽ⁿ⁻¹⁾². For a 3 × 3 matrix, it is |A|⁴.
044: Example: If A is a 3 × 3 matrix and |A| = 2, find |adj(adj(A))|. Answer = 2⁴ = 16.
045: 14. Double Adjoint Matrix Rule
046: Shortcut: adj(adj(A)) = |A|ⁿ⁻² · A.
047: Example: If A = [1 5; 2 7], find adj(adj(A)). Since it's a 2 × 2 matrix, n-2 = 0. Answer = A = [1 5; 2 7].
048: 15. Scalar Adjoint Rule
049: Shortcut: adj(kA) = kⁿ⁻¹ adj(A). For a 3 × 3 matrix, it becomes k² adj(A).
050: Example: If A is a 3 × 3 matrix, find adj(3A). Answer = 9 adj(A).
051: 16. Reverse Cayley-Hamilton (Polynomial Inverse)
052: Shortcut: If A² - 4A + 3I = 0, find A⁻¹ by multiplying the equation by A⁻¹.
053: Example: A - 4I + 3A⁻¹ = 0 ⇒ 3A⁻¹ = 4I - A. Answer: A⁻¹ = 1/3(4I - A).
054: 17. Single Cofactor Check (MCQ Hack)
055: Shortcut: When finding adj(A) from 4 options, calculate ONLY C₁₁ (Row 1, Col 1). Look at the top-left element of the options.
056: Example: A = [3 1 2; 2 -3 5; 5 2 -3]. C₁₁ = (-3)(-3) - (5)(2) = 9-10 = -1. Answer: Choose the option with -1 in the top-left corner.
057: GROUP 3: System of Equations
058: 18. Homogeneous Non-Trivial Hack
059: Shortcut: If ax+by+cz=0, dx+ey+fz=0, gx+hy+iz=0 has a "non-trivial solution," the determinant of coefficients Δ MUST equal 0.
060: Example: x+y+z=0, 2x-y+3z=0, kx+y+2z=0 has non-trivial solutions. Find k. Set |1 1 1; 2 -1 3; k 1 2| = 0. Answer: k = 5/2.
061: 19. The "Linear Combination" Bypass
062: Shortcut: In a homogeneous system, if Equation 1 + Equation 2 matches the x and y coefficients of Equation 3, the determinant is automatically 0.
063: Example: x+2y+z=0, 2x-y+3z=0, 3x+y+λz=0. Eq 1 + Eq 2 gives 3x + y + 4z = 0. To have non-trivial solutions, Eq 3 must be identical. Answer: λ = 4.
064: 20. Non-Homogeneous Infinite Solutions
065: Shortcut: If the system has "infinite solutions," then Δ = 0, Δₓ = 0, Δᵧ = 0, Δz = 0.
066: Example: Use Δ = 0 to quickly find the missing variable before checking the rest.
067: 21. Non-Homogeneous No Solution
068: Shortcut: If the system has "no solution," then Δ = 0, but at least one of Δₓ, Δᵧ, Δz ≠ 0.
069: GROUP 4: Properties of Special Matrices
070: 22. Orthogonal Matrix Inverse
071: Shortcut: If a matrix is orthogonal (AAᵀ = I), its inverse requires ZERO calculation. A⁻¹ = Aᵀ.
072: Example: If A = [cosθ sinθ; -sinθ cosθ], find A⁻¹. Answer: Aᵀ = [cosθ -sinθ; sinθ cosθ].
073: 23. Idempotent Matrix Determinant
074: Shortcut: If A is idempotent (A² = A), its determinant can ONLY be 0 or 1.
075: Example: If a matrix satisfies A² = A and is non-singular, what is |A|? Answer: 1.
076: 24. Involutory Matrix Inverse
077: Shortcut: If A is involutory (A² = I), it is its own inverse. A⁻¹ = A.
078: Example: The Pauli spin matrix σₓ = [0 1; 1 0] squares to I. Answer: Its inverse is [0 1; 1 0].
079: 25. Nilpotent Matrix Trace & Determinant
080: Shortcut: If A is nilpotent (Aᵏ = 0), its determinant is ALWAYS 0 and its Trace is ALWAYS 0.
081: Example: If A = [0 5 6; 0 0 7; 0 0 0], A³ = 0. Answer: |A| = 0 and Trace = 0.
082: 26. Symmetric Product Commutativity
083: Shortcut: If A and B are symmetric matrices, their product AB is symmetric IF AND ONLY IF they commute (AB = BA).
084: Example: True/False: If A and B are symmetric, AB-BA is symmetric. Answer: False. AB-BA is ALWAYS skew-symmetric.
085: GROUP 5: Characteristic Equation & Eigen Hacks
086: 27. The  2 times 2 Characteristic Shortcut
087: Shortcut: Any 2 × 2 matrix A ALWAYS satisfies the equation: A² - (Trace)A + |A|I = 0.
088: Example: A = [4 1; 2 3]. Trace = 4+3=7. |A| = 12-2 = 10. Answer: The equation is A² - 7A + 10I = 0.
089: 28. Trace = Sum of Roots
090: Shortcut: The sum of the roots (eigenvalues) of a matrix is exactly equal to its Trace (sum of principal diagonal elements).
091: Example: A = [5 1 2; 0 3 4; 0 0 -2]. Answer: Sum of roots = 5 + 3 - 2 = 6.
092: 29. Determinant = Product of Roots
093: Shortcut: The product of the roots (eigenvalues) is exactly equal to the Determinant of the matrix (|A|).
094: Example: For an upper triangular matrix, the roots are the diagonal elements. Product = 5 × 3 × (-2) = -30. Answer: |A| = -30
095: 30. Determinant Parameter Isolation (D value hack)
096: Shortcut: If expanding a determinant with a variable λ yields Aλ³ + Bλ² + Cλ + D, you can find the constant term D instantly by plugging λ = 0 into the original determinant.
097: Example: |2-λ 1; 3 4-λ| = λ² - 6λ + D. Find D. Substitute λ=0 ⇒ |2 1; 3 4| = 8 - 3. Answer: D = 5.

107: Chapter 2 - Permutations & Combinations
108: GROUP 1: Words, Numbers & Arrangements
109: 1. The "Sum of All Numbers" Hack (No Zeros)
110: Shortcut: Sum = (n-1)! x (Sum of given digits) x (111... n times).
111: Pattern: "Find the sum of all 4-digit numbers formed by 1, 2, 3, 4 without repetition."
112: Example: n=4. Sum of digits = 1+2+3+4=10.Answer: 3! x 10 x 1111 = 6 x 10 x 1111 = 66660.
113: 2. The "Sum of All Numbers" Hack (With Zero)
114: Shortcut: Sum = [(n-1)! x (Sum) x (111... n times)] - [(n-2)! x (Sum) x (111... n-1 times)].
115: Pattern: "Find the sum of all 4-digit numbers formed by 0, 1, 2, 3."
116: Example: n=4. Sum = 6.Answer: [3! x 6 x 1111] - [2! x 6 x 111] = 39996 - 1332 = 38664.
117: 3. Circular Permutation (People around a table)
118: Shortcut: Exactly (n-1)! ways.
119: Pattern: "In how many ways can 5 people sit around a circular table?"
120: Example: (5-1)! = 4! = 24.
121: 4. Circular Permutation (Garland/Necklace)
122: Shortcut: Exactly (n-1)! / 2 ways. (Because clockwise and anti-clockwise look exactly the same).
123: Pattern: "In how many ways can 6 distinct flowers form a garland?"
124: Example: (6-1)! / 2 = 5! / 2 = 120 / 2 = 60.
125: 5. The "Always Together" Hack (String Method)
126: Shortcut: Tie the items that must be together into 1 single unit. Factorial of total units x Factorial of tied items.
127: Pattern: "Arrange 'MOBILE' so that vowels are always together."
128: Example: Vowels (O, I, E) = 1 unit. Consonants (M, B, L) = 3 units. Total = 4 units.Answer: 4! x 3! = 24 x 6 = 144.
129: 6. Difference of Permutations Formula
130: Shortcut: nPr - (n-1)Pr = r x (n-1)P(r-1).
131: Pattern: "Evaluate 20P3 - 19P3."
132: Example: Answer is simply 3 x 19P2 = 3 x (19 x 18) = 1026.
133: GROUP 2: Factors & Divisors
134: 7. Total Number of Factors
135: Shortcut: Prime factorize the number N = p^a x q^b x r^c. Total factors = (a+1)(b+1)(c+1).
136: Pattern: "Find the total number of divisors of 60."
137: Example: 60 = 2^2 x 3^1 x 5^1.Answer: (2+1)(1+1)(1+1) = 3 x 2 x 2 = 12.
138: Example: 60 = 2^2 x 3^1 x 5^1.Answer: (2+1)(1+1)(1+1) = 3 x 2 x 2 = 12.
139: 8. Number of ODD Factors
140: Shortcut: Prime factorize. Ignore the power of 2 completely. Just add 1 to the other powers and multiply.
141: Pattern: "Find the number of odd divisors of 360."
142: Example: 360 = 2^3 x 3^2 x 5^1. Ignore the 2^3.Answer: (2+1)(1+1) = 3 x 2 = 6.
143: 9. Number of EVEN Factors
144: Shortcut: Do not add 1 to the power of 2. Just use a(b+1)(c+1).
145: Pattern: "Find the number of even divisors of 360."
146: Example: 360 = 2^3 x 3^2 x 5^1.Answer: 3 x (2+1) x (1+1) = 3 x 3 x 2 = 18.
147: 10. Resolving into Two Factors
148: Shortcut: If the number is not a perfect square, the ways to split it into two factors is exactly 1/2 x (Total Factors).
149: Pattern: "In how many ways can 60 be written as a product of two factors?"
150: Example: Total factors of 60 is 12.Answer: 12 / 2 = 6 ways.
151: 11. The Fractional Equation Hack
152: Shortcut: The number of positive integer solutions for 1/x + 1/y = 1/n is exactly the Total number of factors of n^2.
153: Pattern: "Find the positive integral solutions for 1/x + 1/y = 1/10."
154: Example: n^2 = 100 = 2^2 x 5^2.Answer: Factors of 100 = (2+1)(2+1) = 3 x 3 = 9.
155: GROUP 3: Distribution & Equations (Beggar's Method)
156: 12. Non-Negative Integral Solutions
157: Shortcut: For x1 + x2 + ... + xr = n, the number of non-negative solutions (>= 0) is (n+r-1)C(r-1).
158: Pattern: "Find the non-negative integer solutions of x + y + z = 10."
159: Example: n=10, r=3.Answer: (10+3-1)C(3-1) = 12C2 = 66.
160: 13. Positive Integral Solutions
161: Shortcut: For x1 + x2 + ... + xr = n, the number of positive solutions (>= 1) is (n-1)C(r-1).
162: Pattern: "Find the positive integer solutions of x + y + z = 10."
163: Example: n=10, r=3.Answer: (10-1)C(3-1) = 9C2 = 36.
164: 14. The Inequality Solutions Hack
165: Shortcut: For x1 + x2 + ... + xr <= n, introduce a dummy variable. The answer is directly (n+r)Cr.
166: Pattern: "Find the non-negative solutions for x + y + z &lt;= 10."
167: Example: n=10, r=3.Answer: (10+3)C3 = 13C3 = 286.
168: GROUP 4: Geometry Hacks
169: 15. Number of Diagonals
170: Shortcut: For an n-sided polygon, the number of diagonals is n(n-3)/2.
171: Pattern: "Find the number of diagonals in a decagon (10 sides)."
172: Example: n=10. Answer: 10(7)/2 = 35.
173: 16. Handshakes / Lines from points
174: Shortcut: Between n people or points, the number of handshakes or straight lines is nC2 = n(n-1)/2.
175: Pattern: "10 people meet and shake hands. How many handshakes?"
176: Example: Answer: (10 x 9)/2 = 45.
177: 17. Lines with Collinear Points
178: Shortcut: If there are n points and m of them are in a straight line, total lines = nC2 - mC2 + 1.
179: Pattern: "10 points in a plane, 4 are collinear. How many lines can be drawn?"
180: Example: Answer: 10C2 - 4C2 + 1 = 45 - 6 + 1 = 40.
181: 18. Triangles from points
182: Shortcut: From n points, you can make nC3 triangles.
183: Pattern: "How many triangles from 8 non-collinear points?"
184: Example: Answer: 8C3 = 56.
185: 19. Triangles with Collinear Points
186: Shortcut: If m points are collinear out of n, total triangles = nC3 - mC3. (No "+1" here because collinear points cannot make a triangle).
187: Pattern: "10 points in a plane, 4 are collinear. How many triangles?"
188: Example: Answer: 10C3 - 4C3 = 120 - 4 = 116.
189: 20. Intersection of Lines
190: Shortcut: The maximum number of intersection points for n straight lines is nC2.
191: Pattern: "Find the maximum intersection points of 5 straight lines."
192: Example: Answer: 5C2 = 10.
193: 21. Intersection of Circles
194: Shortcut: The maximum number of intersection points for n circles is 2 x nC2 = n(n-1).
195: Pattern: "Find maximum intersection points of 5 circles."
196: Example: Answer: 5 x 4 = 20.
197: 22. Maximum Regions by Lines
198: Shortcut: The maximum number of regions a plane is divided into by n lines is n(n+1)/2 + 1.
199: Pattern: "Into how many maximum parts can 4 lines cut a paper?"
200: Example: Answer: (4 x 5)/2 + 1 = 10 + 1 = 11.
201: GROUP 5: Grids, Boards & Derangements
202: 23. Rectangles in a Grid (Chessboard)
203: Shortcut: In an n x n grid, the number of rectangles is (n(n+1)/2)^2.
204: Pattern: "How many rectangles are on a standard 8 x 8 chessboard?"
205: Example: n=8. Answer: ((8 x 9)/2)^2 = 36^2 = 1296.
206: 24. Squares in a Grid (Chessboard)
207: Shortcut: In an n x n grid, the number of squares is n(n+1)(2n+1)/6 (Sum of squares of first n numbers).
208: Pattern: "How many squares are on a standard 8 x 8 chessboard?"
209: Example: n=8. Answer: (8 x 9 x 17)/6 = 204.
210: 25. Shortest Grid Pathways
211: Shortcut: The number of shortest paths from (0,0) to (m,n) on a grid is (m+n)Cm.
212: Pattern: "Find the number of shortest paths from (0,0) to (3,4)."
213: Example: m=3, n=4. Answer: (3+4)C3 = 7C3 = 35.
214: 26. The Derangement Memorization
215: Shortcut: Derangements mean "nothing goes in its correct place" (like letters in wrong envelopes). Do not calculate it. Just memorize: D3 = 2, D4 = 9, D5 = 44.
216: Pattern: "4 letters are put into 4 envelopes randomly. Ways that ALL are in the wrong envelope?"
217: Example: Answer: D4 = 9.
218: 27. Selection from IDENTICAL items
219: Shortcut: To select at least 1 item from p identical items of one kind, q of another, and r of another: (p+1)(q+1)(r+1) - 1.
220: Pattern: "Ways to select at least one fruit from 3 identical apples and 2 identical bananas."
221: Example: Answer: (3+1)(2+1) - 1 = 12 - 1 = 11.
222: 28. Selection from DISTINCT items
223: Shortcut: To select at least 1 item from n distinctly different items: 2^n - 1.
224: Pattern: "Ways a student can pass an exam having 5 distinct subjects."
225: Example: Answer: 2^5 - 1 = 32 - 1 = 31.
226: 29. Exactly One Pair of Twin Items
227: Shortcut: Number of ways to select r items from n pairs of shoes, such that there is EXACTLY one pair: nC1 x (n-1)C(r-2) x 2^(r-2).
228: Pattern: "From 5 pairs of shoes, select 4 shoes such that exactly 1 pair is formed."
229: Example: n=5, r=4. Answer: 5C1 x 4C2 x 2^2 = 5 x 6 x 4 = 120.
230: 30. Pascal's Triangle Addition
231: Shortcut: nCr + nC(r-1) = (n+1)Cr. (Increase top index by 1, keep the biggest bottom index).
232: Pattern: "Evaluate 15C6 + 15C5."
233: Example: The top matches (15), the bottom are neighbors (6, 5).Answer: Add 1 to top, keep biggest bottom -> 16C6.
234: GROUP 6: The Dictionary Rank Bypasses
235: Traditional method: Drawing alphabetical trees and counting factorials manually.
236: 31. The "Right-Side Smaller" Hack (Word Rank without repetition)
237: The Shortcut: Write the word. Number letters alphabetically. Above each letter, write how many smaller numbers are to its right. Multiply these from right-to-left by 0!, 1!, 2!, 3!$ etc. Add 1 at the end.
238: Example: Rank of MATH. (Alphabetical: A=1, H=2, M=3, T=4).
239: M(3) A(1) T(4) H(2).
240: Smaller to right: M has 2 (A,H). A has 0. T has 1 (H). H has 0.
241: Multiply: (2 × 3!) + (0 × 2!) + (1 × 1!) + (0 × 0!) = 12 + 0 + 1 + 0 = 13.
242: Answer: 13 + 1 = 14.
243: 32. The "Reverse Dictionary" Hack
244: The Shortcut: If asked for the rank from the bottom (end of dictionary), simply find the normal rank of the word, and subtract it from the (Total possible words + 1).

247: Chapter 3 - Sequence & Series
248: GROUP 1: Arithmetic Progression (A.P.)
249: 1. The "Reverse Term" Hack
250: Shortcut: The r-th term is exactly (m + n - r).
251: Pattern: "If the m-th term of an A.P. is n, and the n-th term is m, find the r-th term."
252: Example: The 10th term is 5, and the 5th term is 10. Find the 15th term. Answer: 10 + 5 - 15 = 0.
253: 2. The "Reverse Sum" Hack
254: Shortcut: The sum of (m + n) terms is always -(m + n).
255: Pattern: "If the sum of m terms of an A.P. is n, and the sum of n terms is m, find the sum of m+n terms."
256: Example: S10 = 20, and S20 = 10. Find S30. Answer: -(10 + 20) = -30.
257: 3. The "Equal Multiples" Hack
258: Shortcut: The (m + n)-th term is always 0.
259: Pattern: "If m times the m-th term of an A.P. is equal to n times the n-th term, find the (m+n)-th term."
260: Example: 5 · T5 = 8 · T8. Find T13. Answer: T13 = 0.
261: 4. The "Equal Sums" Hack
262: Shortcut: The sum of (m + n) terms is always 0.
263: Pattern: "If the sum of the first m terms of an A.P. equals the sum of the first n terms..."
264: Example: If S5 = S10, find S15. Answer: S15 = 0.
265: 5. Ratio of Sums to Ratio of Terms
266: Shortcut: Replace 'n' with (2n - 1) in the given ratio.
267: Pattern: "Given the ratio of the sums of n terms of two A.P.s, find the ratio of their n-th terms."
268: Example: Ratio of sums is (3n + 1) : (2n - 3). Find the ratio of 5th terms. Substitute n = 2(5) - 1 = 9. Answer: (27+1) : (18-3) = 28 : 15.
269: 6. Ratio of Terms to Ratio of Sums
270: Shortcut: Replace 'n' with ½(n + 1) in the given ratio.
271: Pattern: "Given the ratio of the n-th terms of two A.P.s, find the ratio of the sum of their n terms."
272: Example: Ratio of terms is (2n + 1) : (n + 3). Find the ratio of the sum of 7 terms. Substitute n = (7+1)/2 = 4. Answer: (8+1) : (4+3) = 9 : 7.
273: 7. Common Terms of Two A.P.s
274: Shortcut: The common terms form a new A.P. whose common difference is the LCM of the two original common differences.
275: Pattern: "Find the number of common terms between two given A.P.s."
276: Example: A.P. 1 has a common difference of 3. A.P. 2 has a common difference of 4. The new common A.P. will have a difference of LCM(3, 4) = 12.
277: 8. Sum of an Odd Number of Terms
278: Shortcut: Sum = n * (Middle Term).
279: Pattern: "The sum of n terms (where n is odd) of an A.P. is given..."
280: Example: Find the sum of 11 terms of an A.P. if its 6th term (middle term) is 20. Answer: 11 × 20 = 220.
281: 9. The Fractional Terms Hack
282: Shortcut: The (mn)-th term is exactly 1. The sum of (mn) terms is ½(mn + 1).
283: Pattern: "If the p-th term of an A.P. is ¹/q and the q-th term is ¹/p..."
284: Example: T3 = ¼, and T4 = ⅓. Find T12. Answer: 1.
285: 10. Maximum Sum of an A.P.
286: Shortcut: Identify the last positive term in the series. The sum up to that term is the absolute maximum.
287: Pattern: "Find the maximum possible sum of the given A.P. (which starts positive and decreases)."
288: Example: A.P. is 20, 18, 16... The last positive term is 2 (which is the 10th term). The maximum sum is S10.
289: 11. Three Terms in A.P.
290: Shortcut: ALWAYS assume the three terms as (a - d), (a), and (a + d).
291: Pattern: "The sum of three numbers in A.P. is S and their product is P."
292: Example: The sum of three numbers in A.P. is 15. (a-d) + a + (a+d) = 15 --> 3a = 15 --> a = 5. (Instantly finds the middle term).

294: GROUP 2: Geometric Progression (G.P.)
295: 12. Product of 'n' Terms in G.P.
296: Shortcut: Product = (First Term × Last Term)n/2.
297: Pattern: "Find the product of the first n terms of a G.P."
298: Example: Product of 5 terms where the 1st term is 2 and the 5th term is 32. Answer: (2 × 32)5/2 = (64)5/2 = 215.
299: 13. Three Terms in G.P.
300: Shortcut: ALWAYS assume the three terms as (a/r), (a), and (ar).
301: Pattern: "The product of three numbers in G.P. is P."
302: Example: The product of 3 numbers is 216. (a/r) · a · (ar) = a3 = 216 --> a = 6. (Instantly finds the middle term).
303: 14. The Exponent Cancellation Hack
304: Shortcut: The value of xq-r · yr-p · zp-q is ALWAYS exactly 1.
305: Pattern: "If the p-th, q-th, and r-th terms of a G.P. are x, y, z respectively, evaluate..."
306: Example: (Matches pattern exactly). Answer: 1.
307: 15. Logarithms of a G.P.
308: Shortcut: If a, b, c are in G.P., then log(a), log(b), log(c) are ALWAYS in A.P.
309: Pattern: "If a, b, c are in G.P., then log_x(a), log_x(b), log_x(c) are in..."
310: Example: (Matches pattern exactly). Answer: Arithmetic Progression (A.P.).
311: 16. Infinite G.P. Condition Trick
312: Shortcut: S∞ = a / (1 - r). But it is a trap! This formula is ONLY valid if -1 < r < 1.
313: Pattern: "Evaluate the infinite geometric series for x..."
314: Example: If 1 + x + x2 ... = 4, find x. Answer: 1 / (1-x) = 4 --> 1 = 4 - 4x --> x = ¾. (Valid because ¾ is between -1 and 1).
315: 17. Recurring Decimals to Fractions
316: Shortcut: 0.pqr_pqr_pqr... = pqr / 999.
317: Pattern: "Convert the recurring decimal into a fraction using geometric series."
318: Example: Convert 0.123123... into a fraction. Answer: 123 / 999 (which simplifies to 41 / 333).

905: Pattern: Find the term independent of x in the expansion of (axp + b/xq)n.

1453: GROUP 2: Standard Sequences (Instant Formulas)
1454: 4. Variance of First 'n' Natural Numbers
1455: Pattern: Find the variance or SD of the numbers 1, 2, 3, ... n.
1456: Shortcut: The variance is ALWAYS exactly (n² - 1) / 12.
1457: Example: Find the variance of the first 11 natural numbers. Here n=11. Answer: (11² - 1) / 12 = 120 / 12 = 10.
1458: 5. Variance of an Arithmetic Progression (A.P.)
1459: Pattern: Find the variance of a sequence of 'n' terms in an A.P. with a common difference 'd'.
1460: Shortcut: The variance is ALWAYS exactly [d² * (n² - 1)] / 12.
1461: Example: Find the variance of 3, 6, 9, 12, 15. Here n=5, d=3. Answer: [3² * (5² - 1)] / 12 = [9 * 24] / 12 = 18.
1462: 6. Variance of First 'n' Even/Odd Numbers
1463: Pattern: Find the variance of the first 'n' EVEN natural numbers or first 'n' ODD natural numbers.
1464: Shortcut: Both have the exact same variance! It is ALWAYS exactly (n² - 1) / 3.
1465: Example: Find the variance of the first 7 even natural numbers. Here n=7. Answer: (7² - 1) / 3 = 48 / 3 = 16.
1466: 7. The "Two Number" SD Hack
1467: Pattern: Find the Standard Deviation of exactly two numbers 'a' and 'b'.
1468: Shortcut: Do not use the long variance formula. The SD is ALWAYS exactly |a - b| / 2 (half of their absolute difference).
1469: Example: Find the SD of 14 and 20. Answer: |14 - 20| / 2 = 6 / 2 = 3.
1470: GROUP 3: Theoretical Minima & Maxima (No Calculations)
1471: 8. The Absolute Deviation Minimum
1472: Pattern: Find the value of 'A' for which the sum of absolute deviations Sum|xᵢ - A| is minimum.
1473: Shortcut: The sum of absolute deviations is ALWAYS minimum when measured about the MEDIAN.
1474: Example: (Matches pattern). Answer: A = Median.
1475: 9. The Squared Deviation Minimum
1476: Pattern: Find the value of 'A' for which the sum of squared deviations Sum(xᵢ - A)² is minimum.
1477: Shortcut: The sum of squared deviations is ALWAYS minimum when measured about the MEAN.
1478: Example: For a dataset, Sum(xᵢ - A)² is minimum. What is A? Answer: The Arithmetic Mean.
1479: 10. The Maximum Variance Hack
1480: Pattern: Find the maximum possible variance of a dataset where all values are strictly bounded in the interval [a, b].
1481: Shortcut: The absolute maximum variance is ALWAYS exactly (b - a)² / 4.
1482: Example: The marks of 50 students lie between 20 and 80. Find the maximum possible variance. Here a=20, b=80. Answer: (80 - 20)² / 4 = 3600 / 4 = 900.

1484: GROUP 4: Corrections & Combinations
1485: 11. The Correct Mean Hack
1486: Pattern: Correcting the mean when an incorrect observation was recorded.
1487: Shortcut: Correct Mean = Old Mean + (Right Value - Wrong Value) / n.
1488: Example: Mean of 20 observations is 50. One observation 30 was wrongly recorded as 80. Correct Mean = 50 + (30 - 80) / 20 = 50 - (50/20) = 50 - 2.5 = 47.5.
1489: 12. Combined Variance (Same Mean)
1490: Pattern: Find the combined variance of two groups that happen to have the EXACT SAME mean.
1491: Shortcut: Do not use the massive d1, d2 formula. The combined variance is simply the weighted average: (n1V1 + n2V2) / (n1 + n2).
1492: Example: Group A (n=10, V=4, Mean=50) and Group B (n=20, V=7, Mean=50). Answer: (104 + 207) / 30 = (40 + 140) / 30 = 180 / 30 = 6.
1493: 13. The Consistency / Stability Checker
1494: Pattern: A word problem asks which of two datasets (like batsmans' scores or factory wages) is more "consistent", "stable", or "uniform".
1495: Shortcut: Find the Coefficient of Variation (CV) = (SD / Mean) * 100. The dataset with the LESSER CV is ALWAYS the more consistent/stable one.
1496: Example: (Matches pattern). Answer: Calculate CV for both, pick the smaller one.
1497: GROUP 5: Standard Distribution Ratios
1498: 14. The 4:5:6 Ratio Hack (Symmetric Distributions)
1499: Pattern: Find the relationship between Quartile Deviation (QD), Mean Deviation (MD), and Standard Deviation (SD) for a normal or symmetric distribution.
1500: Shortcut: Memorize the ratio 4:5:6. This means QD / 4 = MD / 5 = SD / 6.
1501: Example: If SD = 12, find QD. QD / 4 = 12 / 6 -> QD / 4 = 2 -> QD = 8.
1502: 15. The Empirical Formula Trick
1503: Pattern: Find Mean, Median, or Mode when the other two are given.
1504: Shortcut: Mode = 3*(Median) - 2*(Mean). (Memorize it as "Mode = 3 Med - 2 Mean").
1505: Example: In a moderately skewed distribution, Mean = 20, Median = 22. Find Mode. Answer: Mode = 3(22) - 2(20) = 66 - 40 = 26.
