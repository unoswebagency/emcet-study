Formula Sheet
Quick reference for all important formulas

Algebra

1. Functions

  • Domain and Range: For \(\sqrt{f(x)}\), \(f(x) \geq 0\). For \(\frac{1}{f(x)}\), \(f(x) \neq 0\). For \(\log(f(x))\), \(f(x) > 0\).
  • Composite Functions: \((g \circ f)(x) = g(f(x))\).
  • Inverse Functions: \(f(f^{-1}(x)) = x\). A function must be bijective (one-to-one and onto) to have an inverse.
  • Types: Even: \(f(-x) = f(x)\). Odd: \(f(-x) = -f(x)\).

2. Mathematical Induction

  • Sum of first \(n\) natural numbers: \(\sum n = \frac{n(n+1)}{2}\).
  • Sum of squares: \(\sum n^2 = \frac{n(n+1)(2n+1)}{6}\).
  • Sum of cubes: \(\sum n^3 = \left[\frac{n(n+1)}{2}\right]^2\).

3. Matrices

  • Determinant (\(2 \times 2\)): \(|A| = ad - bc\).
  • Inverse: \(A^{-1} = \frac{\text{adj}(A)}{|A|}\).
  • Adjoint Properties:
    \(A(\text{adj } A) = (\text{adj } A)A = |A|I\).
    \(|\text{adj } A| = |A|^{n-1}\).
    \(\text{adj}(AB) = \text{adj}(B)\text{adj}(A)\).
  • System of Equations (Cramer's Rule): \(x = \frac{\Delta_1}{\Delta}, y = \frac{\Delta_2}{\Delta}, z = \frac{\Delta_3}{\Delta}\).

4. Complex Numbers

  • Form: \(z = a + ib\); \(|z| = \sqrt{a^2+b^2}\).
  • Argument: \(\theta = \tan^{-1}(b/a)\).
  • Cube Roots of Unity: \(1, \omega, \omega^2\).
    \(1 + \omega + \omega^2 = 0\).
    \(\omega^3 = 1\).
  • Square root of \(a+ib\): \(\pm \left[ \sqrt{\frac{|z|+a}{2}} + i\text{sgn}(b)\sqrt{\frac{|z|-a}{2}} \right]\).

5. De Moivre’s Theorem

  • Theorem: \((\cos \theta + i\sin \theta)^n = \cos(n\theta) + i\sin(n\theta)\).
  • \(n\)-th roots of unity: \(z_k = e^{i\frac{2k\pi}{n}}\) for \(k = 0, 1, ..., n-1\).
    Sum of \(n\)-th roots \(= 0\).
    Product of \(n\)-th roots \(= (-1)^{n-1}\).

6. Quadratic Expressions

  • Roots: \(x = \frac{-b \pm \sqrt{b^2-4ac}}{2a}\).
  • Nature of Roots (\(\Delta = b^2-4ac\)):
    \(\Delta > 0\): Real and distinct.
    \(\Delta = 0\): Real and equal.
    \(\Delta < 0\): Complex conjugates.
  • Maximum/Minimum Value: For \(ax^2+bx+c\):
    If \(a > 0\), Min value is \(\frac{4ac-b^2}{4a}\) at \(x = -b/2a\).
    If \(a < 0\), Max value is \(\frac{4ac-b^2}{4a}\) at \(x = -b/2a\).

7. Theory of Equations

  • Relation between roots and coefficients (Cubic): \(ax^3+bx^2+cx+d=0\)
    \(\alpha + \beta + \gamma = -b/a\).
    \(\alpha\beta + \beta\gamma + \gamma\alpha = c/a\).
    \(\alpha\beta\gamma = -d/a\).
  • Condition for Arithmetic Progression (AP): \(2b^3 - 9abc + 27a^2d = 0\).

8. Permutations & Combinations

  • Permutations: \(^nP_r = \frac{n!}{(n-r)!}\).
  • Combinations: \(^nC_r = \frac{n!}{r!(n-r)!}\).
  • Circular Permutation: \((n-1)!\).
  • Key Identity: \(^nC_r + ^nC_{r-1} = ^{n+1}C_r\).

9. Binomial Theorem

  • General Term: \(T_{r+1} = ^nC_r x^{n-r} a^r\).
  • Middle Term:
    If \(n\) is even: \((\frac{n}{2} + 1)^{th}\) term.
    If \(n\) is odd: \((\frac{n+1}{2})^{th}\) and \((\frac{n+3}{2})^{th}\) terms.
  • Sum of Coefficients: Put all variables \(= 1\).

10. Partial Fractions

  • Linear Factors: \(\frac{1}{(x-a)(x-b)} = \frac{A}{x-a} + \frac{B}{x-b}\).
  • Repeated Factors: \(\frac{1}{(x-a)^2} = \frac{A}{x-a} + \frac{B}{(x-a)^2}\).
  • Quadratic Factors: \(\frac{1}{ax^2+bx+c} = \frac{Ax+B}{ax^2+bx+c}\).

💡 Quick Algebra Strategy for EAMCET:

Substitution is King: In Algebra, 40% of questions can be solved by substituting \(n=1, 2\) or \(x=1, 0\). This is especially true for Matrices, Induction, and Binomial Theorem. If the options contain \(n\), always test with a small number first!

Trigonometry

1. Trigonometric Ratios

  • Basic Identities:
    \(\sin^2\theta + \cos^2\theta = 1\)
    \(\sec^2\theta - \tan^2\theta = 1\)
    \(\csc^2\theta - \cot^2\theta = 1\)
  • Compound Angles:
    \(\sin(A \pm B) = \sin A \cos B \pm \cos A \sin B\)
    \(\cos(A \pm B) = \cos A \cos B \mp \sin A \sin B\)
    \(\tan(A \pm B) = \frac{\tan A \pm \tan B}{1 \mp \tan A \tan B}\)
  • Multiple & Submultiple Angles:
    \(\sin 2\theta = 2\sin\theta\cos\theta = \frac{2\tan\theta}{1+\tan^2\theta}\)
    \(\cos 2\theta = \cos^2\theta - \sin^2\theta = 2\cos^2\theta - 1 = 1 - 2\sin^2\theta = \frac{1-\tan^2\theta}{1+\tan^2\theta}\)
    \(\tan 2\theta = \frac{2\tan\theta}{1-\tan^2\theta}\)
    \(\sin 3\theta = 3\sin\theta - 4\sin^3\theta\)
    \(\cos 3\theta = 4\cos^3\theta - 3\cos\theta\)
  • Transformations:
    \(\sin C + \sin D = 2\sin(\frac{C+D}{2})\cos(\frac{C-D}{2})\)
    \(\cos C - \cos D = 2\sin(\frac{C+D}{2})\sin(\frac{D-C}{2})\)

2. Trigonometric Equations

  • General Solutions:
    If \(\sin \theta = \sin \alpha \implies \theta = n\pi + (-1)^n \alpha\)
    If \(\cos \theta = \cos \alpha \implies \theta = 2n\pi \pm \alpha\)
    If \(\tan \theta = \tan \alpha \implies \theta = n\pi + \alpha\)
    If \(\sin^2 \theta = \sin^2 \alpha \implies \theta = n\pi \pm \alpha\)

3. Inverse Trigonometric Functions

  • Domain & Range:
    \(\sin^{-1}x \in [-\pi/2, \pi/2]\)
    \(\cos^{-1}x \in [0, \pi]\)
    \(\tan^{-1}x \in (-\pi/2, \pi/2)\)
  • Properties:
    \(\sin^{-1}x + \cos^{-1}x = \pi/2\)
    \(\tan^{-1}x + \tan^{-1}y = \tan^{-1}(\frac{x+y}{1-xy})\) (if \(xy < 1\))
    \(\sin^{-1}(-x) = -\sin^{-1}x\); but \(\cos^{-1}(-x) = \pi - \cos^{-1}x\)

4. Hyperbolic Functions

  • Definitions:
    \(\sinh x = \frac{e^x - e^{-x}}{2}\)
    \(\cosh x = \frac{e^x + e^{-x}}{2}\)
  • Identities:
    \(\cosh^2x - \sinh^2x = 1\)
    \(\text{sech}^2x + \tanh^2x = 1\)
    \(\sinh(x \pm y) = \sinh x \cosh y \pm \cosh x \sinh y\)

5. Properties of Triangles

  • Sine Rule: \(\frac{a}{\sin A} = \frac{b}{\sin B} = \frac{c}{\sin C} = 2R\) (where \(R\) is circumradius).
  • Cosine Rule: \(a^2 = b^2 + c^2 - 2bc \cos A\).
  • Projection Rule: \(a = b \cos C + c \cos B\).
  • Area (\(\Delta\)): \(\Delta = \frac{1}{2}ab \sin C = \sqrt{s(s-a)(s-b)(s-c)}\).
  • Radii:
    \(r = \frac{\Delta}{s}\) (Inradius)
    \(R = \frac{abc}{4\Delta}\) (Circumradius)
    \(r_1 = \frac{\Delta}{s-a}\) (Ex-radius)

💡 Quick Trigonometry Strategy for EAMCET:

Value Picking: For "Identity" type questions (where the equation is true for all angles), replace \(A, B, C\) with \(60^\circ\) or \(x\) with \(45^\circ\). If it's a triangle property, assume the triangle is Equilateral (\(A=B=C=60^\circ\) and \(a=b=c\)) to check the options instantly. This saves you from long theoretical proofs!

Vector Algebra

1. Addition of Vectors

  • Magnitude of a Vector: If \(\vec{a} = x\hat{i} + y\hat{j} + z\hat{k}\), then \(|\vec{a}| = \sqrt{x^2 + y^2 + z^2}\).
  • Unit Vector: \(\hat{a} = \frac{\vec{a}}{|\vec{a}|}\).
  • Direction Cosines (DCs): \(l = \frac{x}{|\vec{a}|}, m = \frac{y}{|\vec{a}|}, n = \frac{z}{|\vec{a}|}\). Remember \(l^2 + m^2 + n^2 = 1\).
  • Section Formula:
    Internal: \(\vec{r} = \frac{m\vec{b} + n\vec{a}}{m+n}\)
    External: \(\vec{r} = \frac{m\vec{b} - n\vec{a}}{m-n}\)
  • Centroid of a Triangle: \(\vec{G} = \frac{\vec{a} + \vec{b} + \vec{c}}{3}\).
  • Centroid of a Tetrahedron: \(\vec{G} = \frac{\vec{a} + \vec{b} + \vec{c} + \vec{d}}{4}\).

2. Product of Vectors (Dot Product)

  • Formula: \(\vec{a} \cdot \vec{b} = |\vec{a}||\vec{b}| \cos \theta\).
  • Condition for Perpendicularity: \(\vec{a} \cdot \vec{b} = 0\).
  • Projection:
    Projection of \(\vec{a}\) on \(\vec{b} = \frac{\vec{a} \cdot \vec{b}}{|\vec{b}|}\).
    Vector projection of \(\vec{a}\) on \(\vec{b} = \left( \frac{\vec{a} \cdot \vec{b}}{|\vec{b}|^2} \right) \vec{b}\).

3. Cross Product (Vector Product)

  • Formula: \(\vec{a} \times \vec{b} = (|\vec{a}||\vec{b}| \sin \theta) \hat{n}\).
  • Condition for Parallelism: \(\vec{a} \times \vec{b} = 0\).
  • Area Formulas:
    Area of Triangle \(= \frac{1}{2} |\vec{a} \times \vec{b}|\) (where \(\vec{a}, \vec{b}\) are adjacent sides).
    Area of Parallelogram \(= |\vec{a} \times \vec{b}|\) (sides) OR \(\frac{1}{2} |\vec{d_1} \times \vec{d_2}|\) (diagonals).

4. Triple Products

  • Scalar Triple Product (STP): \([\vec{a} \vec{b} \vec{c}] = \vec{a} \cdot (\vec{b} \times \vec{c})\).
    Represents the Volume of a Parallelopiped.
    If \([\vec{a} \vec{b} \vec{c}] = 0\), the vectors are Coplanar.
    Volume of Tetrahedron \(= \frac{1}{6} |[\vec{a} \vec{b} \vec{c}]|\).
  • Vector Triple Product (VTP): \(\vec{a} \times (\vec{b} \times \vec{c}) = (\vec{a} \cdot \vec{c})\vec{b} - (\vec{a} \cdot \vec{b})\vec{c}\). (Remember: "BAC - CAB" rule).

💡 Quick Vector Algebra Strategy for EAMCET:

  1. Work with Components: Most EAMCET questions give vectors in \(\hat{i}, \hat{j}, \hat{k}\) form. Use the determinant method for cross products and STP to avoid calculation errors.
  2. Angle Shortcut: If you need to find the angle between two vectors, always use the Dot Product (\(\cos \theta\)) rather than the Cross Product—it is much faster and avoids square roots in the numerator.
  3. The Zero Vector: If the question mentions that vectors form a triangle or a closed loop, remember \(\vec{a} + \vec{b} + \vec{c} = 0\). Using this often simplifies complex product questions.

Coordinate Geometry

1. Basics & Straight Lines

  • Distance Formula: \(d = \sqrt{(x_2-x_1)^2 + (y_2-y_1)^2}\).
  • Section Formula: \(P = \left( \frac{mx_2+nx_1}{m+n}, \frac{my_2+ny_1}{m+n} \right)\).
  • Area of Triangle: \(\frac{1}{2} |x_1(y_2-y_3) + x_2(y_3-y_1) + x_3(y_1-y_2)|\).
  • Slope (\(m\)): \(m = \tan \theta = \frac{y_2-y_1}{x_2-x_1}\).
  • Equations of Line:
    Point-Slope: \(y - y_1 = m(x - x_1)\).
    Intercept Form: \(\frac{x}{a} + \frac{y}{b} = 1\).
    Normal Form: \(x \cos \alpha + y \sin \alpha = p\).
  • Angle between lines: \(\tan \theta = \left| \frac{m_1 - m_2}{1 + m_1m_2} \right|\).
  • Distance from point \((x_1, y_1)\) to line \(ax+by+c=0\): \(d = \frac{|ax_1 + by_1 + c|}{\sqrt{a^2+b^2}}\).

2. Pair of Straight Lines

  • General Equation: \(ax^2 + 2hxy + by^2 = 0\) (passing through origin).
  • Angle: \(\tan \theta = \frac{2\sqrt{h^2-ab}}{|a+b|}\).
  • Condition for Perpendicularity: \(a + b = 0\).
  • Condition for Parallelism: \(h^2 = ab\).

3. Circle & System of Circles

  • Standard Eq: \((x-h)^2 + (y-k)^2 = r^2\).
  • General Eq: \(x^2 + y^2 + 2gx + 2fy + c = 0\).
    Center: \((-g, -f)\).
    Radius: \(\sqrt{g^2+f^2-c}\).
  • Power of Point \(P(x_1, y_1)\): \(S_{11} = x_1^2 + y_1^2 + 2gx_1 + 2fy_1 + c\).
  • Common Tangents:
    Intersecting circles: 2 common tangents.
    Touching externally: 3 common tangents (\(C_1C_2 = r_1+r_2\)).
    Touching internally: 1 common tangent (\(C_1C_2 = |r_1-r_2|\)).
  • Radical Axis: \(S - S' = 0\).

4. Conic Sections (Parabola, Ellipse, Hyperbola)

Feature Parabola (\(y^2=4ax\)) Ellipse (\(\frac{x^2}{a^2} + \frac{y^2}{b^2}=1\)) Hyperbola (\(\frac{x^2}{a^2} - \frac{y^2}{b^2}=1\))
Eccentricity (\(e\)) \(e = 1\) \(e = \sqrt{1-\frac{b^2}{a^2}} < 1\) \(e = \sqrt{1+\frac{b^2}{a^2}} > 1\)
Focus \((a, 0)\) \((\pm ae, 0)\) \((\pm ae, 0)\)
Directrix \(x = -a\) \(x = \pm a/e\) \(x = \pm a/e\)
Latus Rectum \(4a\) \(2b^2/a\) \(2b^2/a\)
Tangency Cond. \(c = a/m\) \(c^2 = a^2m^2 + b^2\) \(c^2 = a^2m^2 - b^2\)

5. Three Dimensional Geometry

  • Distance: \(\sqrt{(x_2-x_1)^2 + (y_2-y_1)^2 + (z_2-z_1)^2}\).
  • Direction Cosines (DCs): \(l, m, n\) where \(l^2+m^2+n^2=1\).
  • Angle between two lines: \(\cos \theta = l_1l_2 + m_1m_2 + n_1n_2\).
  • Plane Equation: \(ax + by + cz + d = 0\).

💡 Quick Coordinate Geometry Strategy for EAMCET:

  1. The "Point" Test: If a question asks for the equation of a line, circle, or conic that passes through a specific point, do not solve the equation. Plug the point into the options. Usually, only one option will satisfy the coordinates.
  2. Tangency Shortcut: Whenever a line \(y=mx+c\) is a tangent to any curve, always use the condition \(p=r\) (perpendicular distance from center equals radius) for circles, or the specific \(c\) formulas for conics.
  3. Locus Problems: For locus questions, pick a simple point that satisfies the given condition and check which option's equation is satisfied by that point.

Calculus

1. Limits and Continuity

  • Standard Limits:
    \(\lim_{x \to 0} \frac{\sin x}{x} = 1\)
    \(\lim_{x \to 0} \frac{e^x - 1}{x} = 1\)
    \(\lim_{x \to 0} \frac{\log(1+x)}{x} = 1\)
    \(\lim_{x \to 0} (1+x)^{1/x} = e\)
  • L’Hopital’s Rule: If \(\lim_{x \to a} \frac{f(x)}{g(x)}\) results in \(\frac{0}{0}\) or \(\frac{\infty}{\infty}\), then:
    \(\lim_{x \to a} \frac{f(x)}{g(x)} = \lim_{x \to a} \frac{f'(x)}{g'(x)}\)
  • Continuity: A function is continuous at \(x=a\) if \(LHL = RHL = f(a)\).

2. Differentiation

  • Chain Rule: \(\frac{d}{dx}[f(g(x))] = f'(g(x)) \cdot g'(x)\).
  • Product Rule: \((uv)' = u'v + uv'\).
  • Quotient Rule: \(\left(\frac{u}{v}\right)' = \frac{vu' - uv'}{v^2}\).
  • Parametric Differentiation: If \(x = f(t), y = g(t)\), then \(\frac{dy}{dx} = \frac{dy/dt}{dx/dt}\).
  • Derivative of Inverse Trig:
    \(\frac{d}{dx}(\sin^{-1}x) = \frac{1}{\sqrt{1-x^2}}\)
    \(\frac{d}{dx}(\tan^{-1}x) = \frac{1}{1+x^2}\)

3. Applications of Derivatives (AOD)

  • Errors and Approximations: \(\Delta y \approx f'(x) \Delta x\).
  • Tangent and Normal:
    Slope of tangent \(m = f'(x_1)\).
    Slope of normal \(m' = -1/f'(x_1)\).
  • Maxima and Minima:
    Find \(f'(x) = 0\) for critical points.
    If \(f''(x) > 0\), it is a Minima.
    If \(f''(x) < 0\), it is a Maxima.

4. Integration (Indefinite)

  • Standard Integrals:
    \(\int \frac{1}{x^2+a^2} dx = \frac{1}{a} \tan^{-1}(\frac{x}{a}) + C\)
    \(\int \frac{1}{\sqrt{a^2-x^2}} dx = \sin^{-1}(\frac{x}{a}) + C\)
    \(\int e^x [f(x) + f'(x)] dx = e^x f(x) + C\)
  • Integration by Parts: \(\int uv dx = u \int v dx - \int [u' \int v dx] dx\).

5. Definite Integrals

  • King's Property: \(\int_a^b f(x) dx = \int_a^b f(a+b-x) dx\). (Very important for EAMCET).
  • Even/Odd Functions: \(\int_{-a}^a f(x) dx = 2\int_0^a f(x) dx\) if \(f(x)\) is even, and \(0\) if \(f(x)\) is odd.
  • Area Under Curve: \(A = \int_a^b y dx\).

6. Differential Equations

  • Variable Separable: \(\int f(y) dy = \int g(x) dx\).
  • Linear Differential Equation: \(\frac{dy}{dx} + Py = Q\).
    Integrating Factor (I.F.): \(e^{\int P dx}\).
    Solution: \(y(I.F.) = \int Q(I.F.) dx + C\).

💡 Quick Calculus Strategy for EAMCET:

  1. Differentiate the Options: For Indefinite Integration, if the question looks too complex to solve, simply differentiate the four options. The one that results in the question's integrand is the answer.
  2. L'Hopital's is your Best Friend: Don't waste time on algebraic expansions for limits. Apply L'Hopital's rule immediately for \(0/0\) forms.
  3. Property 4 (King's Rule): In Definite Integrals, if you see a complicated trig function with limits like \(0\) to \(\pi/2\), \(90\%\) of the time the answer will involve \(\int f(x) + f(\pi/2-x)\).

Dispersion & Probability

1. Measures of Dispersion

  • Mean (\(\bar{x}\)): \(\bar{x} = \frac{\sum x_i}{n}\).
  • Variance (\(\sigma^2\)):
    \(\sigma^2 = \frac{\sum (x_i - \bar{x})^2}{n}\)
    Alternative: \(\sigma^2 = \frac{\sum x_i^2}{n} - (\bar{x})^2\).
  • Standard Deviation (\(\sigma\)): \(\sigma = \sqrt{\text{Variance}}\).
  • Coefficient of Variation (C.V.): \(C.V. = \frac{\sigma}{\bar{x}} \times 100\).
  • Mean Deviation (M.D.):
    About Mean: \(M.D. = \frac{\sum |x_i - \bar{x}|}{n}\).
    About Median: \(M.D. = \frac{\sum |x_i - \text{Median}|}{n}\).

2. Probability

  • Basic Formula: \(P(A) = \frac{\text{Number of favorable outcomes}}{\text{Total number of outcomes}}\).
  • Addition Theorem: \(P(A \cup B) = P(A) + P(B) - P(A \cap B)\).
  • Conditional Probability: \(P(A|B) = \frac{P(A \cap B)}{P(B)}\).
  • Independent Events: \(P(A \cap B) = P(A) \cdot P(B)\).
  • Bayes' Theorem:
    \(P(E_i|A) = \frac{P(E_i)P(A|E_i)}{\sum P(E_j)P(A|E_j)}\).
    (Used when an event has already occurred and you need to find the probability of a specific cause).

3. Random Variables & Distributions

  • Probability Mass Function (PMF): \(\sum P(X=x_i) = 1\).
  • Mean (Expected Value): \(\mu = E(X) = \sum x_i P(x_i)\).
  • Variance: \(V(X) = E(X^2) - [E(X)]^2\), where \(E(X^2) = \sum x_i^2 P(x_i)\).
Binomial Distribution
  • Formula: \(P(X=r) = ^nC_r p^r q^{n-r}\) (where \(q = 1-p\)).
  • Mean: \(np\).
  • Variance: \(npq\).
Poisson Distribution
  • Formula: \(P(X=x) = \frac{e^{-\lambda} \lambda^x}{x!}\).
  • Mean and Variance: Both are equal to \(\lambda\).

💡 Quick Dispersion & Probability Strategy for EAMCET:

  1. Venn Diagrams: For any "Set-based" probability question (e.g., "Probability that at least one of A or B occurs"), draw a quick Venn diagram. It is much harder to make a mistake visually than algebraically.
  2. Property of Variance: Remember that \(Var(ax + b) = a^2 Var(x)\). Adding a constant to every observation does not change the variance, but multiplying every observation by a constant \(a\) changes the variance by \(a^2\).
  3. The "At Least" Trick: When a question asks for the probability of "at least one" success, it is almost always faster to calculate \(1 - P(\text{No success})\).
  4. Symmetry in Distributions: In Binomial distribution questions, if \(p = q = 0.5\), the distribution is symmetric, and the middle term will be the maximum probability.

Mechanics & General Physics

1. Kinematics & Dynamics

  • Equations of Motion:
    \(v = u + at\)
    \(s = ut + \frac{1}{2}at^2\)
    \(v^2 = u^2 + 2as\)
  • Newton's Second Law: \(F = ma = \frac{dp}{dt}\).
  • Work, Energy, Power:
    \(W = \vec{F} \cdot \vec{d} = Fd \cos \theta\)
    \(KE = \frac{1}{2}mv^2\), \(PE = mgh\)

2. Rotational Motion

  • Moment of Inertia: \(I = \sum m_i r_i^2\).
    Ring: \(MR^2\), Disc: \(\frac{1}{2}MR^2\).
  • Torque: \(\vec{\tau} = \vec{r} \times \vec{F} = I\alpha\).
  • Angular Momentum: \(\vec{L} = \vec{r} \times \vec{p} = I\omega\).

Electromagnetism

3. Electrostatics & Circuits

  • Coulomb's Law: \(F = \frac{1}{4\pi\epsilon_0} \frac{q_1q_2}{r^2}\).
  • Electric Potential: \(V = \frac{W}{q}\).
  • Ohm's Law: \(V = IR\).
  • Capacitance: \(C = \frac{Q}{V}\).
    Parallel Plate: \(C = \frac{\epsilon_0 A}{d}\).

4. Magnetism & EMI

  • Magnetic Force: \(F = q(\vec{v} \times \vec{B})\).
  • Faraday's Law: \(\mathcal{E} = -\frac{d\Phi}{dt}\).
  • Transformer: \(\frac{V_s}{V_p} = \frac{N_s}{N_p}\).

Optics & Modern Physics

5. Ray & Wave Optics

  • Mirror/Lens Formula: \(\frac{1}{f} = \frac{1}{v} + \frac{1}{u}\) (Mirror), \(\frac{1}{f} = \frac{1}{v} - \frac{1}{u}\) (Lens).
  • Snell's Law: \(n_1 \sin \theta_1 = n_2 \sin \theta_2\).
  • Fringe Width: \(\beta = \frac{\lambda D}{d}\).

6. Modern Physics

  • Photoelectric Equation: \(h\nu = \Phi + K_{max}\).
  • De Broglie Wavelength: \(\lambda = \frac{h}{p} = \frac{h}{mv}\).
  • Mass-Energy Equivalence: \(E = mc^2\).

💡 Physics Strategy for EAMCET:

  1. Unit Check: Always verify units. If an answer is in Newtons, but your calculation used cm instead of meters, you'll pick the wrong option.
  2. Dimension Analysis: In complex formula-based questions, check the dimensions of the four options. Often, only one option has the correct dimensions for the quantity asked.
  3. Focus on High Weightage: Mechanics and Electromagnetism usually carry the most marks. Master these two first.

Organic & General Chemistry

Chapter 1 - Matrices & Determinants

GROUP 1: Determinant Value Hacks (Zero Calculation)

1. The Arithmetic Progression (A.P.) Hack

  • Shortcut: If the elements of any row or column of a 3 × 3 determinant are in A.P., the determinant is ALWAYS 0.
  • Example: |2 4 6; 1 5 9; 10 20 30|. Look at Row 1 (common diff = 2) and Row 2 (common diff = 4). Answer = 0.

2. The Logarithmic Geometric Progression (G.P.) Hack

  • Shortcut: If the elements of a determinant are log values of terms in a G.P., the determinant is ALWAYS 0.
  • Example: |log 2 log 4 log 8; log 3 log 9 log 27; log 5 log 25 log 125|. (Since 2, 4, 8 is a G.P., their logs form an A.P.). Answer = 0.

3. Odd-Order Skew-Symmetric Hack

  • Shortcut: The determinant of ANY skew-symmetric matrix (Aᵀ = -A) of odd order (3 × 3, 5 × 5) is ALWAYS 0.
  • Example: |0 7 -4; -7 0 5; 4 -5 0|. (Diagonals are 0, and aᵢⱼ = -aⱼᵢ). Answer = 0.

4. Even-Order Skew-Symmetric Hack

  • Shortcut: The determinant of a skew-symmetric matrix of even order (2 × 2, 4 × 4) is ALWAYS a Perfect Square.
  • Example: If asked for the determinant of a 4 × 4 skew-symmetric matrix and the options are (A) 14, (B) 16, (C) 18, (D) 20. Answer = 16 (only perfect square).

5. Maximum Value with {0, 1}

  • Shortcut: The maximum possible value of a 3 × 3 determinant whose elements are strictly 0 or 1 is exactly 2.
  • Example: "What is the maximum value of a determinant formed using only the binary digits 0 and 1?" Answer = 2.

6. Maximum Value with {-1, 1}

  • Shortcut: The maximum possible value of a 3 × 3 determinant whose elements are strictly -1 or 1 is exactly 4.
  • Example: "What is the max value of a 3 × 3 matrix A where aᵢⱼ ∈ {-1, 1}?" Answer = 4.

7. Diagonal/Triangular Product Rule

  • Shortcut: For any upper triangular, lower triangular, or diagonal matrix, the determinant is simply the product of its principal diagonal elements.
  • Example: |5 8 9; 0 2 7; 0 0 -3|. Answer = 5 × 2 × (-3) = -30.

8. The Vandermonde Standard

  • Shortcut: |1 1 1; a b c; a² b² c²| = (a-b)(b-c)(c-a).
  • Example: |1 1 1; 4 5 6; 16 25 36|. Here a=4, b=5, c=6. Answer = (4-5)(5-6)(6-4) = (-1)(-1)(2) = 2.

9. The Circulant Root Hack

  • Shortcut: For |x a a; a x a; a a x| = 0, the roots are ALWAYS -2a, a, a.
  • Example: |x 5 5; 5 x 5; 5 5 x| = 0. Answer: Roots are -10, 5, 5.

10. Cyclic Symmetric Expansion

  • Shortcut: |a b c; b c a; c a b| = -(a³ + b³ + c³ - 3abc).
  • Example: |1 2 3; 2 3 1; 3 1 2|. Answer = -(1³ + 2³ + 3³ - 3(1)(2)(3)) = -(1 + 8 + 27 - 18) = -18.

GROUP 2: Adjoint & Inverse Hacks

11. The 2 times 2 Inverse Flash-Solve

  • Shortcut: For A = [a b; c d], A⁻¹ = 1/(ad-bc) [d -b; -c a]. (Swap main diagonal, flip signs of the other).
  • Example: A = [2 3; 1 4]. Answer: A⁻¹ = 1/5 [4 -3; -1 2].

12. Adjoint Determinant Rule

  • Shortcut: |adj(A)| = |A|ⁿ⁻¹. For a 3 × 3 matrix, it is simply |A|².
  • Example: If A is a 3 × 3 matrix and |A| = 4, find |adj A|. Answer = 4² = 16.

13. Double Adjoint Determinant Rule

  • Shortcut: |adj(adj(A))| = |A|⁽ⁿ⁻¹⁾². For a 3 × 3 matrix, it is |A|⁴.
  • Example: If A is a 3 × 3 matrix and |A| = 2, find |adj(adj(A))|. Answer = 2⁴ = 16.

14. Double Adjoint Matrix Rule

  • Shortcut: adj(adj(A)) = |A|ⁿ⁻² · A.
  • Example: If A = [1 5; 2 7], find adj(adj(A)). Since it's a 2 × 2 matrix, n-2 = 0. Answer = A = [1 5; 2 7].

15. Scalar Adjoint Rule

  • Shortcut: adj(kA) = kⁿ⁻¹ adj(A). For a 3 × 3 matrix, it becomes k² adj(A).
  • Example: If A is a 3 × 3 matrix, find adj(3A). Answer = 9 adj(A).

16. Reverse Cayley-Hamilton (Polynomial Inverse)

  • Shortcut: If A² - 4A + 3I = 0, find A⁻¹ by multiplying the equation by A⁻¹.
  • Example: A - 4I + 3A⁻¹ = 0 ⇒ 3A⁻¹ = 4I - A. Answer: A⁻¹ = 1/3(4I - A).

17. Single Cofactor Check (MCQ Hack)

  • Shortcut: When finding adj(A) from 4 options, calculate ONLY C₁₁ (Row 1, Col 1). Look at the top-left element of the options.
  • Example: A = [3 1 2; 2 -3 5; 5 2 -3]. C₁₁ = (-3)(-3) - (5)(2) = 9-10 = -1. Answer: Choose the option with -1 in the top-left corner.

GROUP 3: System of Equations

18. Homogeneous Non-Trivial Hack

  • Shortcut: If ax+by+cz=0, dx+ey+fz=0, gx+hy+iz=0 has a "non-trivial solution," the determinant of coefficients Δ MUST equal 0.
  • Example: x+y+z=0, 2x-y+3z=0, kx+y+2z=0 has non-trivial solutions. Find k. Set |1 1 1; 2 -1 3; k 1 2| = 0. Answer: k = 5/2.

19. The "Linear Combination" Bypass

  • Shortcut: In a homogeneous system, if Equation 1 + Equation 2 matches the x and y coefficients of Equation 3, the determinant is automatically 0.
  • Example: x+2y+z=0, 2x-y+3z=0, 3x+y+λz=0. Eq 1 + Eq 2 gives 3x + y + 4z = 0. To have non-trivial solutions, Eq 3 must be identical. Answer: λ = 4.

20. Non-Homogeneous Infinite Solutions

  • Shortcut: If the system has "infinite solutions," then Δ = 0, Δₓ = 0, Δᵧ = 0, Δz = 0.
  • Example: Use Δ = 0 to quickly find the missing variable before checking the rest.

21. Non-Homogeneous No Solution

  • Shortcut: If the system has "no solution," then Δ = 0, but at least one of Δₓ, Δᵧ, Δz ≠ 0.

GROUP 4: Properties of Special Matrices

22. Orthogonal Matrix Inverse

  • Shortcut: If a matrix is orthogonal (AAᵀ = I), its inverse requires ZERO calculation. A⁻¹ = Aᵀ.
  • Example: If A = [cosθ sinθ; -sinθ cosθ], find A⁻¹. Answer: Aᵀ = [cosθ -sinθ; sinθ cosθ].

23. Idempotent Matrix Determinant

  • Shortcut: If A is idempotent (A² = A), its determinant can ONLY be 0 or 1.
  • Example: If a matrix satisfies A² = A and is non-singular, what is |A|? Answer: 1.

24. Involutory Matrix Inverse

  • Shortcut: If A is involutory (A² = I), it is its own inverse. A⁻¹ = A.
  • Example: The Pauli spin matrix σₓ = [0 1; 1 0] squares to I. Answer: Its inverse is [0 1; 1 0].

25. Nilpotent Matrix Trace & Determinant

  • Shortcut: If A is nilpotent (Aᵏ = 0), its determinant is ALWAYS 0 and its Trace is ALWAYS 0.
  • Example: If A = [0 5 6; 0 0 7; 0 0 0], A³ = 0. Answer: |A| = 0 and Trace = 0.

26. Symmetric Product Commutativity

  • Shortcut: If A and B are symmetric matrices, their product AB is symmetric IF AND ONLY IF they commute (AB = BA).
  • Example: True/False: If A and B are symmetric, AB-BA is symmetric. Answer: False. AB-BA is ALWAYS skew-symmetric.

GROUP 5: Characteristic Equation & Eigen Hacks

27. The 2 times 2 Characteristic Shortcut

  • Shortcut: Any 2 × 2 matrix A ALWAYS satisfies the equation: A² - (Trace)A + |A|I = 0.
  • Example: A = [4 1; 2 3]. Trace = 4+3=7. |A| = 12-2 = 10. Answer: The equation is A² - 7A + 10I = 0.

28. Trace = Sum of Roots

  • Shortcut: The sum of the roots (eigenvalues) of a matrix is exactly equal to its Trace (sum of principal diagonal elements).
  • Example: A = [5 1 2; 0 3 4; 0 0 -2]. Answer: Sum of roots = 5 + 3 - 2 = 6.

29. Determinant = Product of Roots

  • Shortcut: The product of the roots (eigenvalues) is exactly equal to the Determinant of the matrix (|A|).
  • Example: For an upper triangular matrix, the roots are the diagonal elements. Product = 5 × 3 × (-2) = -30. Answer: |A| = -30

30. Determinant Parameter Isolation (D value hack)

  • Shortcut: If expanding a determinant with a variable λ yields Aλ³ + Bλ² + Cλ + D, you can find the constant term D instantly by plugging λ = 0 into the original determinant.
  • Example: |2-λ 1; 3 4-λ| = λ² - 6λ + D. Find D. Substitute λ=0 ⇒ |2 1; 3 4| = 8 - 3. Answer: D = 5.

Chapter 2 - Permutations & Combinations

GROUP 1: Words, Numbers & Arrangements

1. The "Sum of All Numbers" Hack (No Zeros)

  • Shortcut: Sum = (n-1)! x (Sum of given digits) x (111... n times).
  • Pattern: "Find the sum of all 4-digit numbers formed by 1, 2, 3, 4 without repetition."
  • Example: n=4. Sum of digits = 1+2+3+4=10.Answer: 3! x 10 x 1111 = 6 x 10 x 1111 = 66660.

2. The "Sum of All Numbers" Hack (With Zero)

  • Shortcut: Sum = [(n-1)! x (Sum) x (111... n times)] - [(n-2)! x (Sum) x (111... n-1 times)].
  • Pattern: "Find the sum of all 4-digit numbers formed by 0, 1, 2, 3."
  • Example: n=4. Sum = 6.Answer: [3! x 6 x 1111] - [2! x 6 x 111] = 39996 - 1332 = 38664.

3. Circular Permutation (People around a table)

  • Shortcut: Exactly (n-1)! ways.
  • Pattern: "In how many ways can 5 people sit around a circular table?"
  • Example: (5-1)! = 4! = 24.

4. Circular Permutation (Garland/Necklace)

  • Shortcut: Exactly (n-1)! / 2 ways. (Because clockwise and anti-clockwise look exactly the same).
  • Pattern: "In how many ways can 6 distinct flowers form a garland?"
  • Example: (6-1)! / 2 = 5! / 2 = 120 / 2 = 60.

5. The "Always Together" Hack (String Method)

  • Shortcut: Tie the items that must be together into 1 single unit. Factorial of total units x Factorial of tied items.
  • Pattern: "Arrange 'MOBILE' so that vowels are always together."
  • Example: Vowels (O, I, E) = 1 unit. Consonants (M, B, L) = 3 units. Total = 4 units.Answer: 4! x 3! = 24 x 6 = 144.

6. Difference of Permutations Formula

  • Shortcut: nPr - (n-1)Pr = r x (n-1)P(r-1).
  • Pattern: "Evaluate 20P3 - 19P3."
  • Example: Answer is simply 3 x 19P2 = 3 x (19 x 18) = 1026.

GROUP 2: Factors & Divisors

7. Total Number of Factors

  • Shortcut: Prime factorize the number N = p^a x q^b x r^c. Total factors = (a+1)(b+1)(c+1).
  • Pattern: "Find the total number of divisors of 60."
  • Example: 60 = 2^2 x 3^1 x 5^1.Answer: (2+1)(1+1)(1+1) = 3 x 2 x 2 = 12.
  • Example: 60 = 2^2 x 3^1 x 5^1.Answer: (2+1)(1+1)(1+1) = 3 x 2 x 2 = 12.

    • 8. Number of ODD Factors

    • Shortcut: Prime factorize. Ignore the power of 2 completely. Just add 1 to the other powers and multiply.
    • Pattern: "Find the number of odd divisors of 360."
    • Example: 360 = 2^3 x 3^2 x 5^1. Ignore the 2^3.Answer: (2+1)(1+1) = 3 x 2 = 6.

    9. Number of EVEN Factors

    • Shortcut: Do not add 1 to the power of 2. Just use a(b+1)(c+1).
    • Pattern: "Find the number of even divisors of 360."
    • Example: 360 = 2^3 x 3^2 x 5^1.Answer: 3 x (2+1) x (1+1) = 3 x 3 x 2 = 18.

    10. Resolving into Two Factors

    • Shortcut: If the number is not a perfect square, the ways to split it into two factors is exactly 1/2 x (Total Factors).
    • Pattern: "In how many ways can 60 be written as a product of two factors?"
    • Example: Total factors of 60 is 12.Answer: 12 / 2 = 6 ways.

    11. The Fractional Equation Hack

    • Shortcut: The number of positive integer solutions for 1/x + 1/y = 1/n is exactly the Total number of factors of n^2.
    • Pattern: "Find the positive integral solutions for 1/x + 1/y = 1/10."
    • Example: n^2 = 100 = 2^2 x 5^2.Answer: Factors of 100 = (2+1)(2+1) = 3 x 3 = 9.

    GROUP 3: Distribution & Equations (Beggar's Method)

    12. Non-Negative Integral Solutions

    • Shortcut: For x1 + x2 + ... + xr = n, the number of non-negative solutions (>= 0) is (n+r-1)C(r-1).
    • Pattern: "Find the non-negative integer solutions of x + y + z = 10."
    • Example: n=10, r=3.Answer: (10+3-1)C(3-1) = 12C2 = 66.

    13. Positive Integral Solutions

    • Shortcut: For x1 + x2 + ... + xr = n, the number of positive solutions (>= 1) is (n-1)C(r-1).
    • Pattern: "Find the positive integer solutions of x + y + z = 10."
    • Example: n=10, r=3.Answer: (10-1)C(3-1) = 9C2 = 36.

    14. The Inequality Solutions Hack

    • Shortcut: For x1 + x2 + ... + xr <= n, introduce a dummy variable. The answer is directly (n+r)Cr.
    • Pattern: "Find the non-negative solutions for x + y + z <= 10."
    • Example: n=10, r=3.Answer: (10+3)C3 = 13C3 = 286.

    GROUP 4: Geometry Hacks

    15. Number of Diagonals

    • Shortcut: For an n-sided polygon, the number of diagonals is n(n-3)/2.
    • Pattern: "Find the number of diagonals in a decagon (10 sides)."
    • Example: n=10. Answer: 10(7)/2 = 35.

    16. Handshakes / Lines from points

    • Shortcut: Between n people or points, the number of handshakes or straight lines is nC2 = n(n-1)/2.
    • Pattern: "10 people meet and shake hands. How many handshakes?"
    • Example: Answer: (10 x 9)/2 = 45.

    17. Lines with Collinear Points

    • Shortcut: If there are n points and m of them are in a straight line, total lines = nC2 - mC2 + 1.
    • Pattern: "10 points in a plane, 4 are collinear. How many lines can be drawn?"
    • Example: Answer: 10C2 - 4C2 + 1 = 45 - 6 + 1 = 40.

    18. Triangles from points

    • Shortcut: From n points, you can make nC3 triangles.
    • Pattern: "How many triangles from 8 non-collinear points?"
    • Example: Answer: 8C3 = 56.

    19. Triangles with Collinear Points

    • Shortcut: If m points are collinear out of n, total triangles = nC3 - mC3. (No "+1" here because collinear points cannot make a triangle).
    • Pattern: "10 points in a plane, 4 are collinear. How many triangles?"
    • Example: Answer: 10C3 - 4C3 = 120 - 4 = 116.

    20. Intersection of Lines

    • Shortcut: The maximum number of intersection points for n straight lines is nC2.
    • Pattern: "Find the maximum intersection points of 5 straight lines."
    • Example: Answer: 5C2 = 10.

    21. Intersection of Circles

    • Shortcut: The maximum number of intersection points for n circles is 2 x nC2 = n(n-1).
    • Pattern: "Find maximum intersection points of 5 circles."
    • Example: Answer: 5 x 4 = 20.

    22. Maximum Regions by Lines

    • Shortcut: The maximum number of regions a plane is divided into by n lines is n(n+1)/2 + 1.
    • Pattern: "Into how many maximum parts can 4 lines cut a paper?"
    • Example: Answer: (4 x 5)/2 + 1 = 10 + 1 = 11.

    GROUP 5: Grids, Boards & Derangements

    23. Rectangles in a Grid (Chessboard)

    • Shortcut: In an n x n grid, the number of rectangles is (n(n+1)/2)^2.
    • Pattern: "How many rectangles are on a standard 8 x 8 chessboard?"
    • Example: n=8. Answer: ((8 x 9)/2)^2 = 36^2 = 1296.

    24. Squares in a Grid (Chessboard)

    • Shortcut: In an n x n grid, the number of squares is n(n+1)(2n+1)/6 (Sum of squares of first n numbers).
    • Pattern: "How many squares are on a standard 8 x 8 chessboard?"

    1. The "Reverse Term" Hack

    GROUP 1: The "King's Rule" Bypasses

    1. The Universal (b-a)/2 Hack

    • Shortcut: The answer is ALWAYS exactly (b - a) / 2.
    • Pattern: ∫ₐᵇ [f(x) / (f(x) + f(a + b - x))] dx. (The numerator is one of the terms in the denominator).

    • Shortcut: The answer is ALWAYS exactly π/4, regardless of the power n.
    • Pattern: ∫₀^{π/2} [sinⁿ x / (sinⁿ x + cosⁿ x)] dx (Also works for tan/cot and sec/csc).
    • Example: ∫₀^{π/2} [tan²⁰²⁴ x / (1 + tan²⁰²⁴ x)] dx. (This is just sin/cos disguised). Answer: π/4.

    • Shortcut: The answer is ALWAYS exactly π/12.
    • Pattern: ∫_{π/6}^{π/3} [1 / (1 + √tan x)] dx (or cot x, or any power of them).
    • Example: Notice a + b = π/6 + π/3 = π/2. Using the (b - a) / 2 hack: (π/3 - π/6) / 2 = (π/6) / 2 = π/12.

    • Shortcut: The integral evaluates to exactly 0.
    • Pattern: ∫ₐᵇ log[x / (a + b - x)] dx.
    • Example: ∫₄⁶ log[x / (10 - x)] dx. By King's Rule, the numerator and denominator flip, making it -log, which cancels the original. Answer: 0.

    5. The f(x) ⋅ f(a + b - x) = 1 Hack

    • Shortcut: If f(x) ⋅ f(a + b - x) = 1, then ∫ₐᵇ [1 / (1 + f(x))] dx is exactly (b - a) / 2.

    GROUP 2: Symmetric Limits (-a to a)

    6. The Odd Function Annihilator

    • Shortcut: The answer is ALWAYS exactly 0.
    • Pattern: ∫₋ₐᵃ f(x) dx where f(-x) = -f(x). (Look for sin x, x³, x cos x, or log[(a-x)/(a+x)]).
    • Example: ∫₋₅⁵ x² sin x dx. x² is even, sin x is odd. Even × Odd = Odd. Answer: 0.

    • Shortcut: The answer is ALWAYS exactly 0.
    • Pattern: ∫₋ₐᵃ log[(k - x) / (k + x)] dx.
    • Example: ∫₋₂² log[(5 - x) / (5 + x)] dx. If you put -x, the fraction flips, creating a negative log. It's an odd function. Answer: 0.

    8. The $1 + a^x$ Denominator Eraser

    • Shortcut: The 1 + aˣ denominator completely disappears. The integral becomes just ∫₀ᵃ f(x) dx.
    • Pattern: ∫₋ₐᵃ [f(x) / (1 + kˣ)] dx where f(x) is an EVEN function.
    • Example: ∫₋ₚᵢᵖⁱ [cos² x / (1 + 2ˣ)] dx. Since cos² x is even, drop the denominator and change limits: ∫₀ᵖⁱ cos² x dx = π/2.

    9. The "Constant Sum" Hack (From previous papers)

    • Shortcut: If f(x) ⋅ f(-x) = k², the integral is EXACTLY a/k.

    10. The Periodic Lobe (Mod Sine/Cos)

    • Shortcut: The area of one "lobe" of |sin x| or |cos x| from 0 to π is exactly 2. For n half-periods, the answer is 2n.

    11. The Fractional Part Hack

    • Shortcut: The integral of the fractional part function {x} from 0 to n (integer) is exactly n/2.
    • Pattern: ∫₀ⁿ {x} dx.
    • Example: ∫₀⁵⁰ {x} dx. Answer: 50 / 2 = 25.

    12. The Greatest Integer Function (GIF) Step Hack

    • Shortcut: The integral of [x] from 0 to n (integer) is the sum of the first n-1 integers: n(n-1)/2.
    • Pattern: ∫₀ⁿ [x] dx.
    • Example: ∫₀¹⁰ [x] dx. Answer: (10 × 9) / 2 = 45.

    13. The Shifted Period Bypass

    • Shortcut: You can completely ignore the starting point 'a'. The answer is simply n × ∫₀ᵀ f(x) dx.
    • Pattern: ∫ₐᵃ⁺ⁿᵀ f(x) dx, where T is the period of f(x).
    • Example: ∫₁₀¹⁰⁺⁶ᵖⁱ |sin x| dx. The period of |sin x| is π. Ignore the 10. You just have 6 periods. Answer: 6 × 2 = 12.

    14. Modulus V-Shape Area

    • Shortcut: The answer is exactly ((c - a)² + (b - c)²) / 2.
    • Pattern: ∫ₐᵇ |x - c| dx (where c is between a and b).

    GROUP 4: Wallis’ Formula (0 to pi/2 limits)

    15. Wallis Sine/Cosine (Odd Power)

    • Shortcut: Keep subtracting 2 from the power until you hit 1 (for numerator) and 2 (for denominator). Multiply the fractions.
    • Pattern: ∫₀ᵖⁱ/² sinⁿ x dx where n is an ODD number.
    • Example: ∫₀ᵖⁱ/² sin⁵ x dx. Numerator: 4 × 2. Denominator: 5 × 3 × 1. Answer: 8/15.

    16. Wallis Sine/Cosine (Even Power)

    • Shortcut: Same as #15, but you MUST multiply the final answer by π/2.
    • Pattern: ∫₀ᵖⁱ/² cosⁿ x dx where n is an EVEN number.
    • Example: ∫₀ᵖⁱ/² cos⁶ x dx. Fraction: (5 × 3 × 1) / (6 × 4 × 2) = 15/48 = 5/16. Multiply by π/2. Answer: 5π/32.

    17. Wallis Product (Both Powers Even)

    • Shortcut: Apply Wallis to both powers on top. On the bottom, add the powers, then subtract 2 until you hit 2. Since both are even, multiply by π/2.
    • Pattern: ∫₀ᵖⁱ/² sinᵐ x cosⁿ x dx (where m, n are even).
    • Example: ∫₀ᵖⁱ/² sin² x cos⁴ x dx. Numerator: (1) × (3 × 1) = 3. Denominator: (6 × 4 × 2) = 48. Multiply by π/2. Answer: (3/48) × π/2 = π/32.

    18. Wallis Product (One or Both Odd)

    • Shortcut: Same process as #17, but do NOT multiply by π/2. Just leave the fraction as it is.
    • Pattern: ∫₀ᵖⁱ/² sinᵐ x cosⁿ x dx (where at least one is odd).
    • Example: ∫₀ᵖⁱ/² sin³ x cos² x dx. Numerator: (2) × (1) = 2. Denominator: (5 × 3 × 1) = 15. Answer: 2/15.

    GROUP 1: Functional Equation "Instant Guess" Hacks

    1. The "Additive to Multiplicative" Hack

    • Pattern: If f(x + y) = f(x) ⋅ f(y).

    2. The "Multiplicative to Additive" Hack

    • Pattern: If f(x ⋅ y) = f(x) + f(y).
    • Shortcut: The function is ALWAYS logarithmic: f(x) = k ⋅ log(x).
    • Example: If f(xy) = f(x) + f(y) and f(e) = 1, find f(e2). Since f(e) = k ⋅ log(e) = 1, k = 1. So f(x) = log(x). Answer: f(e2) = 2.

    3. The "Additive to Additive" Hack

    • Pattern: If f(x + y) = f(x) + f(y).
    • Shortcut: The function is ALWAYS a straight line through the origin: f(x) = kx.
    • Example: If f(x+y) = f(x) + f(y) and f(2) = 10, find f(5). Since f(2) = 2k = 10, k = 5. So f(x) = 5x. Answer: f(5) = 25.

    4. The "Multiplicative to Multiplicative" Hack

    • Pattern: If f(x ⋅ y) = f(x) ⋅ f(y).
    • Shortcut: The function is ALWAYS a power function: f(x) = xn.
    • Example: If f(xy) = f(x)f(y) and f(2) = 8, find f(3). Since f(2) = 2n = 8, n = 3. So f(x) = x3. Answer: f(3) = 33 = 27.

    5. The Polynomial Identity Bypass

    • Pattern: If f(x) ⋅ f(1/x) = f(x) + f(1/x), and f(x) is a polynomial.
    • Shortcut: The function is ALWAYS exactly f(x) = 1 + xn OR f(x) = 1 - xn.
    • Example: If f(x)f(1/x) = f(x) + f(1/x) and f(3) = 28, find f(4). Try 1 + xn: 1 + 3n = 28 → 3n = 27 → n = 3. So f(x) = 1 + x3. Answer: f(4) = 1 + 43 = 65.

    GROUP 2: Inverse Function Hacks

    6. The "Linear Fraction" Inverse Hack

    • Pattern: Find the inverse of f(x) = (ax + b) / (cx + d).
    • Shortcut: Swap 'a' and 'd', and change their signs. Inverse is f-1(x) = (-dx + b) / (cx - a).
    • Example: Find the inverse of f(x) = (2x + 3) / (5x - 4). Swap 2 and -4, change signs: 4 and -2. Answer: (4x + 3) / (5x - 2).

    7. The "Self-Inverse" Spotter

    • Pattern: Find the inverse of a fraction where a = -d, like f(x) = (3x + 1) / (2x - 3).
    • Shortcut: If a = -d in the fraction (ax+b)/(cx+d), the function is its OWN inverse. f-1(x) = f(x).
    • Example: Here a=3, d=-3. They are negative opposites. Answer: f-1(x) is exactly (3x + 1) / (2x - 3).

    8. The Inverse Intersection Rule

    • Pattern: Find the intersection point of f(x) and its inverse f-1(x).
    • Shortcut: The curves of f(x) and f-1(x) ALWAYS intersect exactly on the line y = x. Do not equate f(x) = f-1(x). Just equate f(x) = x.
    • Example: Find intersection of f(x) = 3x - 4 and its inverse. Set 3x - 4 = x -> 2x = 4 -> x = 2. Since it's on y=x, y is also 2. Answer: (2, 2).

    9. The Exponential-Log Inverse Flip

    • Pattern: Find the inverse of an exponential like f(x) = 10x+5.
    • Shortcut: The inverse of f(x) = ax is always f-1(x) = loga(x). Don't solve it manually.
    • Example: Let y = 10x+5. Take log base 10: log10(y) = x + 5. Answer: f-1(x) = log(x) - 5.

    10. The Quadratic Inverse Bypass

    • Pattern: Find the inverse of a quadratic function like ax2 + bx + c (with restricted domain).
    • Shortcut: Complete the square to a(x-h)2 + k. The inverse is always h + √((x-k)/a).
    • Example: f(x) = x2 - 4x + 5 (for x > 2). Complete square: (x-2)2 + 1. Here h=2, k=1, a=1. Answer: 2 + √(x-1).

    11. The Inner Minimum Hack (Logarithms)

    • Pattern: Check if log(x + √(x2 + a2)) is an even or odd function.

    • Pattern: Find the even part or odd part of a general function like f(x) = ex.
    • Shortcut: ANY function can be written as the sum of an even and odd function. Even part = (f(x) + f(-x)) / 2. Odd part = (f(x) - f(-x)) / 2.

    • Pattern: If f(x) = x3 ⋅ sin(x), is the total function even or odd?

    26. The Even Power Many-One Rule

    • Pattern: Classify an even-degree polynomial like f(x) = x4 - 2x2 + 3 from R to R.
    • Shortcut: ANY polynomial function with an EVEN highest degree (quadratics, quartics) from R to R is ALWAYS Many-One and Into. (Never One-One, Never Onto).

    27. The Odd Power Onto Rule

    • Pattern: Classify an odd-degree polynomial like f(x) = x3 + 2x2 - x + 5 from R to R.
    • Shortcut: ANY polynomial function with an ODD highest degree from R to R is ALWAYS Onto (Surjective).
    • Example: Highest power is 3 (odd). Answer: Yes, it is Onto.

    28. Total Number of Functions

    • Pattern: Set A has 'm' elements, Set B has 'n' elements. How many total functions exist from A to B?
    • Shortcut: The total number of functions from A to B is exactly nm.
    • Example: Set A has 3 elements, Set B has 4 elements. Here m=3, n=4. Answer: 43 = 64.

    29. Total Number of One-One Functions (Injections)

    • Pattern: Set A has 'm' elements, Set B has 'n' elements. How many One-One functions exist?
    • Shortcut: If m > n, answer is 0. If m <= n, the answer is exactly nPm (Permutations of n taken m at a time).
    • Example: Set A has 3 elements, Set B has 5 elements. Here m=3, n=5. Since 3 < 5, use 5P3. Answer: 5 * 4 * 3 = 60.

    30. Total Number of Bijections

    • Pattern: Set A has 'm' elements, Set B has 'n' elements. How many bijections exist?
    • Shortcut: Bijective functions (One-One AND Onto) can ONLY exist if set A and set B have the EXACT same number of elements (m = n). If they do, the answer is always n!.
    • Example: Set A has 4 elements, Set B has 4 elements. Here m = n = 4. Answer: 4! = 24.

    • Pattern: Evaluate fractions containing (1+i) and (1-i) in the numerator and denominator.
    • Shortcut: ½(1+i)/(1-i) is EXACTLY 'i'. And ½(1-i)/(1+i) is EXACTLY '-i'.
    • Example: Evaluate (½(1+i)/(1-i))100. This simplifies instantly to i100. Since 100 is a multiple of 4, Answer: 1.

    4. The Factorial Power Trap

    • Pattern: Evaluate in! (where n is a factorial).
    • Shortcut: For any n ≥ 4, the factorial n! is a multiple of 4. Therefore, in! is ALWAYS 1 for any n ≥ 4.
    • Example: Find i5!. Since 5! = 120 (a multiple of 4), Answer: 1.

    GROUP 2: Cube Roots of Unity (1, ω, ω2)

    5. The Massive ω Power Hack

    • Pattern: Evaluate huge powers of ω, like ωn.
    • Shortcut: Since ω3 = 1, just divide the power 'n' by 3 and find the remainder (R). The answer is ωR. (If remainder is 0, answer is 1).
    • Example: Find ω2024. Sum of digits of 2024 is 8. 8 divided by 3 gives remainder 2. Answer: ω2.

    6. The Alternating Sign Sum Trap

    • Pattern: Evaluate polynomial expressions containing sums and differences of 1, ω, and ω2.
    • Shortcut: The sum 1 + ω + ω2 is ALWAYS 0. Use this to substitute: ω2 = -1 - ω, or ω = -1 - ω2, or 1+ω = -ω2.
    • Example: Find (1 - ω + ω2)5. Group the positive terms: (1 + ω2) - ω. Substitute (1 + ω2) with -ω. The expression becomes (-ω - ω)5 = (-2ω)5 = -32ω5. Since ω5 = ω2, Answer: -32ω2.

    7. The a3 + b3 Factorization Hack

    • Pattern: Factorizing sums of cubes in the complex plane.
    • Shortcut: a3 + b3 = (a + b)(a + bω)(a + bω2).
    • Example: Find the roots of x3 + 8 = 0. Rewrite as x3 + 23 = 0. The roots are instantly: -2, -2ω, -2ω2.

    8. The a3 - b3 Factorization Hack

    • Pattern: Factorizing differences of cubes in the complex plane.
    • Shortcut: a3 - b3 = (a - b)(a - bω)(a - bω2).
    • Example: Find the roots of x3 - 1 = 0. The roots are instantly: 1, ω, ω2.

    9. The Special Quadratic Root Spotter

    • Pattern: Finding the roots or evaluating expressions based on x2 + x + 1 = 0 or x2 - x + 1 = 0.
    • Shortcut: The roots of x2 + x + 1 = 0 are EXACTLY ω and ω2. The roots of x2 - x + 1 = 0 are EXACTLY -ω and -ω2.
    • Example: If x2 + x + 1 = 0, find x31 + x32. Put x = ω. This becomes ω31 + ω32 = ω1 + ω2. Since 1+ω+ω2 = 0, Answer: -1.

    GROUP 3: De Moivre's Theorem Hacks

    10. The Cosine Addition Hack

    • Pattern: Given z + (1/z) = 2cos(θ), find zn + (1/zn).
    • Shortcut: The answer is ALWAYS 2cos(n * θ).
    • Example: If z + 1/z = 2cos(30°), find z3 + 1/z3. Answer: 2cos(3 * 30°) = 2cos(90°) = 0.

    11. The Sine Subtraction Hack

    • Pattern: Given z + (1/z) = 2cos(θ), find zn - (1/zn).
    • Shortcut: The answer is ALWAYS 2i * sin(n * θ).
    • Example: If z + 1/z = 2cos(30°), find z3 - 1/z3. Answer: 2i * sin(90°) = 2i.

    12. The cis(theta) Division Trick

    • Pattern: Division of complex numbers given in polar form: (cos A + i sin A) / (cos B + i sin B).
    • Shortcut: cis(A) / cis(B) is EXACTLY cis(A - B). Just subtract the angles.
    • Example: cis(75) / cis(15). Answer: cis(75 - 15) = cis(60) = cos(60) + i sin(60) = ½ + i(√(3)/2).

    13. The De Moivre "Wrong Order" Trap

    • Pattern: Expanding powers where sine comes first: (sin A + i cos A)n.
    • Shortcut: De Moivre's theorem ONLY works for (cos A + i sin A). To fix it, swap them by using (90 - A). Answer: cos[n(90 - A)] + i sin[n(90 - A)].
    • Example: Expand (sin 30 + i cos 30)3. Change to cis(90 - 30)3 = cis(60)3. Apply power: cis(180) = cos(180) + i sin(180). Answer: -1.

    GROUP 4: Modulus & Conjugate Bypasses

    14. The Parallelogram Law (Sum to Difference)

    • Pattern: Given |z1 + z2| and asked for |z1 - z2| (or vice versa).
    • Shortcut: |z1 + z2|2 + |z1 - z2|2 = 2( |z1|2 + |z2|2 ).
    • Example: If |z1|=1, |z2|=1, and |z1+z2| = √(2). Find |z1-z2|. (√(2))2 + |z1-z2|2 = 2(1 + 1) -> 2 + X2 = 4 -> X2 = 2. Answer: √(2).

    15. The Unimodular Fraction Hack

    • Pattern: Evaluating the modulus of | (z - a) / (1 - a*z) | where the modulus of z is 1.
    • Shortcut: If |z| = 1 and 'a' is a real number, the modulus of this entire specific fraction is ALWAYS exactly 1.
    • Example: If |z| = 1, find | (z - 5) / (1 - 5z) |. Answer: 1.

    16. Max/Min Value of |z + a|

    • Pattern: Finding the maximum or minimum value of |z + a| when |z| = r is given.
    • Shortcut: The maximum value is |z| + |a|. The minimum value is | |z| - |a| |. (Triangle Inequality).
    • Example: If |z| = 4, find the max value of |z + 3i|. Here |3i| = 3. Max = 4 + 3. Answer: 7.

    17. The Modulus Squared Substitution

    • Pattern: Simplifying expressions with z multiplied by its conjugate (z̄).
    • Shortcut: z * z̄ is EXACTLY equal to |z|2.
    • Example: If |z| = 5, evaluate z * z̄. Answer: 52 = 25.

    18. The Purely Real/Imaginary Conjugate Check

    • Pattern: The question states a complex number is "purely real" or "purely imaginary."
    • Shortcut: If purely real, z = z̄. If purely imaginary, z = -z̄.
    • Example: If z is purely imaginary, find z + z̄. Answer: 0.

    GROUP 5: Geometry & Locus Spotters

    19. The Perpendicular Bisector Locus

    • Pattern: Find the locus of z when |(z - z1) / (z - z2)| = 1, or |z - z1| = |z - z2|.
    • Shortcut: The locus is ALWAYS the perpendicular bisector of the line segment joining points z1 and z2.
    • Example: |z - 2| = |z - 4|. This means distance from x=2 equals distance from x=4. Locus is the vertical line x = 3.

    20. The Circle Diameter Locus

    • Pattern: Find the locus of z when arg((z - z1) / (z - z2)) = +π/2 or -π/2.
    • Shortcut: The locus is ALWAYS a circle with z1 and z2 acting as the endpoints of the diameter.
    • Example: arg((z-1)/(z+1)) = π/2. Locus is a circle with diameter endpoints at (1,0) and (-1,0).

    21. The Ellipse Locus Rule

    • Pattern: Find the locus of z when |z - z1| + |z - z2| = 2a.
    • Shortcut: If 2a is GREATER than the distance between z1 and z2 (|z1 - z2|), the locus is ALWAYS an Ellipse.
    • Example: |z - 3| + |z + 3| = 10. Distance between 3 and -3 is 6. Since 10 > 6, Answer: Ellipse.

    22. The Hyperbola Locus Rule

    • Pattern: Find the locus of z when |z - z1| - |z - z2| = 2a.
    • Shortcut: If 2a is LESS than the distance between z1 and z2 (|z1 - z2|), the locus is ALWAYS a Hyperbola.
    • Example: |z - 5| - |z + 5| = 8. Distance between 5 and -5 is 10. Since 8 < 10, Answer: Hyperbola.

    23. The Standard Circle Locus

    • Pattern: Find the locus or area of |z - z0| = r.
    • Shortcut: This is ALWAYS a circle with its center at z0 and a radius of r.
    • Example: |z - (2+3i)| = 5. Answer: Circle with center (2,3) and radius 5.

    24. Equilateral Triangle with Origin

    • Pattern: A geometric condition where z1, z2, and the origin (0,0) form an equilateral triangle.
    • Shortcut: If they form an equilateral triangle with the origin, the exact relationship is ALWAYS: z12 + z22 - z1*z2 = 0.
    • Example: (Matches pattern). Answer: z12 + z22 = z1*z2.

    25. General Equilateral Triangle Rule

    • Pattern: A geometric condition where any three points z1, z2, z3 form an equilateral triangle.
    • Shortcut: The relationship is ALWAYS: z12 + z22 + z32 = z1z2 + z2z3 + z3*z1.
    • Example: (Matches pattern).

    GROUP 6: Argument (Angle) Hacks

    26. Argument of Pure Numbers

    • Pattern: Find the argument of a purely real or purely imaginary number.
    • Shortcut: Purely positive real -> arg = 0. Purely negative real -> arg = π. Purely positive imaginary -> arg = π/2. Purely negative imaginary -> arg = -π/2.
    • Example: Find arg(-5). It is purely negative real. Answer: π.

    27. The Logarithmic Argument Rule

    • Pattern: Finding the argument of products (z1*z2) or quotients (z1/z2).
    • Shortcut: Treat "arg" exactly like "log". arg(z1 * z2) = arg(z1) + arg(z2). arg(z1 / z2) = arg(z1) - arg(z2).
    • Example: Find arg((1+i)/(1-i)). arg(1+i) = 45°. arg(1-i) = -45°. Answer: 45° - (-45°) = 90°.

    28. Argument of a Conjugate

    • Pattern: Finding the argument of z̄ when arg(z) is known.
    • Shortcut: arg(z̄) is EXACTLY the negative of arg(z). arg(z̄) = -arg(z).
    • Example: If arg(z) = 30°, find arg(z̄). Answer: -30°.

    29. Argument of a Power

    • Pattern: Finding the argument of zn.
    • Shortcut: arg(z^n) = n * arg(z). (Again, exactly like logarithms).
    • Example: Find arg((1+i)4). arg(1+i) = π/4. Answer: 4 * (π/4) = π.

    GROUP 7: Roots of Equations

    30. The "Roots of Unity" Sum/Product Hack

    • Pattern: Finding the sum or product of all n-th roots of unity (1, ω, ω2... ωn-1).
    • Shortcut: The SUM of all n-th roots of unity is ALWAYS exactly 0. The PRODUCT of all n-th roots of unity is (-1)n-1.
    • Example: Find the sum of the 5th roots of unity. Answer: 0. Find the product of the 5th roots of unity. Answer: (-1)^4 = 1.

    Chapter 8 - Binomial Theorem

    GROUP 1: The "r" Index Bypasses (Finding Specific Terms)

    1. The Independent Term Hack

    • Pattern: Find the term independent of x in the expansion of (axp + b/xq)n.
    • Shortcut: You do not need to expand. The index 'r' is ALWAYS exactly: r = (np) / (p + q). If r is a fraction, no independent term exists (answer is 0). The term is T_(r+1).
    • Example: Find the independent term in (x2 - 1/x)9. Here n=9, p=2, q=1. r = (9*2) / (2+1) = 18/3 = 6. The term is T7. Answer: 9C6 * (1)3 * (-1)6 = 9C6.

    2. The Target Power Hack

    • Pattern: Find the coefficient of xm in the expansion of (axp + b/xq)n.
    • Shortcut: The index 'r' is ALWAYS exactly: r = (np - m) / (p + q).
    • Example: Find the coefficient of x4 in (x2 + 1/x)5. Here n=5, p=2, q=1, m=4. r = (5*2 - 4) / (2+1) = 6/3 = 2. The term is T3. Answer: 5C2 * (1)3 * (1)2 = 10.

    3. The Term from the End Hack

    • Pattern: Find the k-th term from the end of the expansion (x + y)n.
    • Shortcut: The k-th term from the end is EXACTLY the (n - k + 2)-th term from the beginning.
    • Example: Find the 4th term from the end in (x + y)^10. Here n=10, k=4. Term from beginning = 10 - 4 + 2 = 8. Answer: Find the 8th term (T_8).

    GROUP 2: Number of Terms Bypasses

    4. The Twin Bracket Addition Hack

    • Pattern: Find the number of terms in the expansion of (x + a)n + (x - a)n.
    • Shortcut: If n is EVEN, the number of terms is (n/2) + 1. If n is ODD, the number of terms is (n+1)/2.
    • Example: Find the number of terms in (x + a)100 + (x - a)100. Since 100 is even, Answer: (100/2) + 1 = 51 terms.

    5. The Twin Bracket Subtraction Hack

    • Pattern: Find the number of terms in the expansion of (x + a)n - (x - a)n.
    • Shortcut: If n is EVEN, the number of terms is n/2. If n is ODD, the number of terms is (n+1)/2.
    • Example: Find the number of terms in (x + a)100 - (x - a)100. Since 100 is even, Answer: 100/2 = 50 terms.

    6. The Multinomial Terms Hack

    • Pattern: Find the total number of terms in the expansion of (a + b + c + ... k terms)n.
    • Shortcut: The number of terms is ALWAYS exactly n + k - 1Ck - 1.
    • Example: Find the number of terms in (x + y + z)10. Here n=10, k=3 (three variables). Answer: 12C2 = 66 terms.

    GROUP 3: Substitution Bypasses (Sum of Coefficients)

    7. Total Sum of Coefficients

    • Pattern: Find the sum of all coefficients in a massive expansion like (ax + by + cz)n.
    • Shortcut: Ignore the expansion completely. Put ALL variables (x, y, z) equal to 1, and simply evaluate the numbers.
    • Example: Find the sum of coefficients in (2x - 3y + 4z)10. Put x=y=z=1. The bracket becomes (2 - 3 + 4) = 3. Answer: 310.

    8. Sum of Even/Odd Place Coefficients

    • Pattern: Find the sum of coefficients of only the EVEN powers of x, or only the ODD powers of x, in f(x) = (ax + b)n.
    • Shortcut: Sum of even powers = [f(1) + f(-1)] / 2. Sum of odd powers = [f(1) - f(-1)] / 2.
    • Example: Find the sum of coefficients of even powers in (1 + x)10. f(1) = 210. f(-1) = 0. Answer: (210 + 0) / 2 = 29.

    GROUP 4: Standard Binomial Coefficient Series (C0, C1, C2...)

  • (Note: Cr means nCr)

    • 9. The Universal Sum Hack

    • Pattern: Evaluate C0 + C1 + C2 + ... + Cn.
    • Shortcut: The sum of all basic binomial coefficients is ALWAYS exactly 2n.
    • Example: Evaluate 10C0 + 10C1 + ... + 10C10. Answer: 210 = 1024.

    10. The Alternate Sum Hack

    • Pattern: Evaluate C0 + C2 + C4 + ... OR C1 + C3 + C5 + ...
    • Shortcut: The sum of just the even-placed, or just the odd-placed binomial coefficients is ALWAYS 2n-1.
    • Example: Evaluate 10C1 + 10C3 + 10C5 + 10C7 + 10C9. Answer: 210-1 = 29 = 512.

    11. The Alternating Sign Hack

    • Pattern: Evaluate C0 - C1 + C2 - C3 + ... +/- Cn.
    • Shortcut: Any full series of binomial coefficients with perfectly alternating + and - signs ALWAYS equals 0.
    • Example: Evaluate 5C0 - 5C1 + 5C2 - 5C3 + 5C4 - 5C5. Answer: 0.

    12. The Multiplier Hack

    • Pattern: Evaluate 1C1 + 2C2 + 3C3 + ... + nCn.
    • Shortcut: The answer is ALWAYS exactly n * 2n-1.
    • Example: Evaluate 1*(10C1) + 2*(10C2) + ... + 10*(10C10). Answer: 10 * 29 = 5120.

    13. The Divider Hack

    • Pattern: Evaluate C0 + C1/2 + C2/3 + ... + Cn/(n+1).
    • Shortcut: The answer is ALWAYS exactly (2n+1 - 1) / (n + 1).
    • Example: Evaluate 4C0 + 4C1/2 + 4C2/3 + 4C3/4 + 4C4/5. Here n=4. Answer: (25 - 1) / 5 = 31 / 5.

    14. The Sum of Squares Hack

    • Pattern: Evaluate (C0)2 + (C1)2 + (C2)2 + ... + (Cn)2.
    • Shortcut: The answer is ALWAYS exactly 2nCn.
    • Example: Evaluate (5C0)2 + (5C1)2 + ... + (5C5)2. Answer: 10C5 = 252.

    15. The Shifted Product Hack

    • Pattern: Evaluate C0Cr + C1Cr+1 + C2*Cr+2 + ... + Cn-r*Cn.
    • Shortcut: The answer is ALWAYS exactly 2nCn - r.
    • Example: Evaluate 10C0*10C2 + 10C1*10C3 + ... + 10C8*10C10. Here n=10, r=2. Answer: 20C8.

    GROUP 5: Greatest Terms & Middle Terms

    16. The Middle Term Spotter

    • Pattern: Find the middle term of (x + y)n.
    • Shortcut: If n is EVEN, there is ONE middle term: T_(n/2 + 1). If n is ODD, there are TWO middle terms: T_((n+1)/2) and T_((n+3)/2).
    • Example: Middle term of (x + y)^8. n is even. Position is (8/2)+1 = 5. Answer: 5th term (T_5).

    17. Greatest Binomial Coefficient

    • Pattern: Find the greatest coefficient in (1 + x)^n.
    • Shortcut: The greatest coefficient ALWAYS belongs to the middle term(s). For even n, it is nCn/2.
    • Example: Find the maximum coefficient in (1 + x)10. Answer: 10C5 = 252.

    18. Numerically Greatest Term (Value)

    • Pattern: Find the numerically greatest term in (1 + a)n for a given value of a.
    • Shortcut: Calculate m = [ (n + 1) * |a| ] / [ 1 + |a| ]. If m is an integer, both T_m and T_(m+1) are the greatest terms. If m is a decimal, the greatest term is T_([m] + 1) where [m] is the integer part.
    • Example: Find greatest term in (1 + 2x)8 when x=1. Here a = 2. m = (9 * 2) / (1 + 2) = 18/3 = 6. Since 6 is an integer, Answer: Both T6 and T7 are the greatest terms.

    GROUP 6: Negative and Fractional Index Bypasses

  • (Only valid when |x| < 1. Number of terms is ALWAYS infinite).

    • 19. Negative Index Coefficient Spotter

    • Pattern: Find the coefficient of xr in the expansion of (1 - x)-n.
    • Shortcut: The coefficient is ALWAYS exactly n + r - 1Cr. All coefficients are positive.
    • Example: Find the coefficient of x4 in (1 - x)-3. Here n=3, r=4. Answer: 6C4 = 15.

    20. The (1 - x)^(-1) Series

    • Pattern: Expansion of 1 / (1 - x) or (1 - x)-1.
    • Shortcut: The coefficient of EVERY term x^r is exactly 1.
    • Example: Find the coefficient of x100 in (1 - x)-1. Answer: 1.

    21. The (1 - x)^(-2) Series

    • Pattern: Expansion of 1 / (1 - x)2 or (1 - x)-2.
    • Shortcut: The coefficient of x^r is ALWAYS exactly (r + 1).
    • Example: Find the coefficient of x50 in (1 - x)-2. Answer: 50 + 1 = 51.

    22. The First Negative Term Hack

    • Pattern: Find the first negative term in the expansion of (1 + x)p/q where p/q is a positive fraction.
    • Shortcut: Check the sign of the multiplier (p/q - r + 1). The first integer 'r' that makes this multiplier negative points to the first negative term T_(r+1).
    • Example: First negative term in (1 + x)7/2. We need (7/2 - r + 1) < 0 -> 4.5 < r. The first integer is r=5. Answer: The 6th term (T6).

    GROUP 7: Remainders and Divisibility (Number Theory Hacks)

    23. The "Last Two Digits ending in 1" Hack

    • Pattern: Find the last two digits of a huge power ending in 1, like (31)14.
    • Shortcut: The last digit is ALWAYS 1. The tens digit is the last digit of (Base's Tens Digit * Power's Last Digit).
    • Example: Find the last two digits of 3114. Last digit = 1. Tens digit = 3 * 4 = 12, keep the 2. Answer: 21.

    24. The Last Digit Hack (Cyclicity)

    • Pattern: Find the last digit of a massive power like 72025.
    • Shortcut: Powers repeat in cycles of 4. Divide the power by 4 to find the remainder R. The last digit is (Base's last digit)^R. (If remainder is 0, use power 4).
    • Example: Find last digit of 72025. 2025 / 4 gives remainder 1. Answer: 71 = 7.

    25. Binomial Remainder Bypass

    • Pattern: Find the remainder when (1 + x)n is divided by x or x2.
    • Shortcut: Never expand fully. Use only the first two terms: 1 + nx. All other terms have x^2 or higher and will be perfectly divisible.
    • Example: Find the remainder when 9100 is divided by 8. Rewrite as (1 + 8)100. Divide by 8. First term is 1. All other terms contain 8 and are perfectly divisible. Answer: 1.

    26. The Difference Divisibility Rule

    • Pattern: Determine what divides xn - yn.
    • Shortcut: x^n - y^n is ALWAYS perfectly divisible by (x - y) for all integer values of n.
    • Example: Is 158 - 48 divisible by 11? Since 15 - 4 = 11, Answer: Yes.

    27. The Even Addition Divisibility Rule

    • Pattern: Determine what divides xn + yn when n is an ODD number.
    • Shortcut: x^n + y^n is ALWAYS perfectly divisible by (x + y) ONLY when n is ODD.
    • Example: Is 127 + 57 divisible by 17? Since n=7 is odd and 12+5 = 17, Answer: Yes.

    GROUP 8: Advanced Binomial Spotters

    28. The Pascal's Triangle Addition Rule

    • Pattern: Simplify the sum of two adjacent combinations: nCr + nCr - 1.
    • Shortcut: Add 1 to the top number 'n', and KEEP the larger bottom number 'r'. Answer: n+1Cr.
    • Example: Evaluate 15C6 + 15C5. The tops match, the bottoms are adjacent. Add 1 to top, keep 6. Answer: 16C6.

    29. The nCr Ratio Hack

    • Pattern: Evaluate the ratio of adjacent binomial coefficients: nCr / nCr - 1.
    • Shortcut: You do not need factorials. The answer is ALWAYS exactly (n - r + 1) / r.
    • Example: Evaluate 20C8 / 20C7. Here n=20, r=8. Answer: (20 - 8 + 1) / 8 = 13 / 8.

    30. The Rational Terms Hack (Irrational Expansions)

    • Pattern: Find the total number of rational (integer) terms in (x1/p + y1/q)n.
    • Shortcut: Find the LCM of the denominators p and q. Divide the total power 'n' by that LCM. If n is perfectly divisible by the LCM, the answer is (n / LCM) + 1.
    • Example: Find the number of rational terms in (21/3 + 31/4)12. LCM of 3 and 4 is 12. Total power n=12. 12 / 12 = 1. Answer: 1 + 1 = 2 rational terms.

    23. Maximum Positive Real Roots (Descartes' Rule)

    • Pattern: Given p + iq is a root of an equation with real coefficients.

    • Pattern: Given p + √q is a root of an equation with rational coefficients.
    • Shortcut: p - √q is automatically also a root. The quadratic factor representing these two roots is ALWAYS x2 - 2px + (p2 - q).
    • Example: 3 + √5 is a root. Here p=3, q=5. The hidden quadratic factor is x2 - 6x + (9-5) = x2 - 6x + 4.

    • Pattern: Find the condition for exactly one root of f(x) = 0 to be zero.

    • Example: For ax3+bx2+cx+d=0, if two roots are exactly 0, then Answer: d = 0 and c = 0.

    GROUP 6: Miscellaneous Equation Hacks

    29. Sum of Coefficients Zero Hack

    • Pattern: You notice the sum of all coefficients in a polynomial equation f(x) = 0 is exactly 0.

    • Pattern: Find the maximum or minimum value of f(x) = asin(x) + bcos(x) + c.

    GROUP 1: Vector Triple Products & Identities

    1. The BAC-CAB Rule (Vector Triple Product)

    • Pattern: Expand a vector triple product like a x (b x c).
    • Shortcut: ALWAYS expands to (a . c)b - (a . b)c. (Memorize it as "BAC minus CAB").
    • Example: Expand i x (j x i). Here a=i, b=j, c=i. Answer: (i . i)j - (i . j)i = (1)j - (0)i = j.

    2. The Lagrange Identity Bypass

    • Pattern: Find |a x b| when magnitudes |a|, |b|, and the dot product (a . b) are known.
    • Shortcut: |a x b|² + (a . b)² = |a|² * |b|².
    • Example: If |a|=2, |b|=3, and a.b = 4. Find |a x b|. Answer: |a x b|² + 16 = (4 * 9). |a x b|² = 36 - 16 = 20. |a x b| = √20.

    3. The Cyclic Scalar Triple Product (STP)

    • Pattern: Evaluate or compare scalar triple products like [a b c], [b c a], [a c b].
    • Shortcut: Cyclic shifts keep the sign: [a b c] = [b c a] = [c a b]. Anti-cyclic shifts flip the sign: [a c b] = -[a b c].
    • Example: Evaluate [i j k] + [j k i] + [k j i]. Answer: 1 + 1 + (-1) = 1.

    • Shortcut: ALWAYS write it directly as: x/a + y/b + z/c = 1.
    • Example: Plane cuts x, y, z axes at 2, 3, 4. Answer: x/2 + y/3 + z/4 = 1.

    22. Family of Intersecting Planes

    • Pattern: Equation of a plane passing through the intersection line of Plane1 = 0 and Plane2 = 0.
    • Shortcut: The equation is ALWAYS of the form: Plane1 + k * Plane2 = 0. Plug in the given third point to find 'k'.
    • Example: (Matches pattern).

    GROUP 6: Lines (3D Geometry)

    23. Angle Between Line and Plane (The Trap)

    • Pattern: Find the angle (theta) between a line (direction a,b,c) and a plane (normal p,q,r).
    • Shortcut: Use SINE, not cosine! sin(theta) = |ap + bq + cr| / (|direction| * |normal|).
    • Example: (Matches pattern).

    24. Line Lying in a Plane Check

    • Pattern: Verify if a given line lies entirely inside a given plane.
    • Shortcut: Two conditions must be met: 1) The point from the line equation must satisfy the plane equation. 2) The dot product of the line's direction and the plane's normal must be 0.
    • Example: (Matches pattern).

    25. Coplanar Lines Determinant Rule

    • Pattern: Find if two lines (x-x1)/a1 = ... and (x-x2)/a2 = ... are coplanar.
    • Shortcut: The 3x3 determinant MUST be 0. Row 1: (x2-x1, y2-y1, z2-z1). Row 2: (a1, b1, c1). Row 3: (a2, b2, c2).
    • Example: (Matches pattern).

    26. Shortest Distance Between Parallel Lines

    • Pattern: Find the distance between lines r = a1 + tb and r = a2 + sb.
    • Shortcut: Distance = |(a2 - a1) x b| / |b|.
    • Example: (Matches pattern).

    27. Shortest Distance Between Skew Lines

    • Pattern: Find the distance between non-parallel, non-intersecting lines r = a1 + tb1 and r = a2 + sb2.
    • Shortcut: Distance = |[ (a2 - a1) b1 b2 ]| / |b1 x b2|. (Numerator is the scalar triple product).
    • Example: (Matches pattern).

    GROUP 7: Points & Ratios (3D Co-ordinate Geometry)

    28. The 3D Coordinate Plane Divider

    • Pattern: Find the ratio in which the XY, YZ, or ZX plane divides the line segment joining (x1, y1, z1) and (x2, y2, z2).
    • Shortcut: XY plane ratio is ALWAYS exactly -z1 / z2. YZ plane is -x1 / x2. ZX plane is -y1 / y2.
    • Example: Ratio in which XY plane divides line joining (1, 2, 3) and (4, 5, -6). Answer: -(3) / (-6) = 1:2.

    29. Centroid of a Tetrahedron

    • Pattern: Find the centroid of a tetrahedron with vertices A, B, C, D.
    • Shortcut: The centroid is simply the average: (A + B + C + D) / 4.
    • Example: Vertices are (1,0,0), (0,1,0), (0,0,1), (0,0,0). Answer: (1/4, 1/4, 1/4).

    30. The "Sphere Center/Radius" Spotter

    • Pattern: Find the center and radius of a sphere from its general equation: x² + y² + z² + 2ux + 2vy + 2wz + d = 0.
    • Shortcut: Center is ALWAYS (-u, -v, -w). Divide x,y,z coefficients by -2. Radius is √(u² + v² + w² - d).
    • Example: Sphere x² + y² + z² - 4x + 6y - 8z + 4 = 0. Center = (2, -3, 4). Radius = √(4 + 9 + 16 - 4) = 5.

    Chapter 12 - Measures of Dispersion

    GROUP 1: The "Change of Origin & Scale" Bypasses

    1. The "Addition/Subtraction" Bypass (Origin)

    • Pattern: Find the new Variance, Standard Deviation (SD), Range, or Mean Deviation when a constant 'k' is added to or subtracted from every observation.
    • Shortcut: These measures of dispersion NEVER CHANGE. They are 100% independent of change of origin.
    • Example: The variance of 10 observations is 5. If 3 is added to every observation, find the new variance. Answer: 5 (It remains completely unchanged).

    2. The "Multiplication" Bypass (Scale)

    • Pattern: Find the new SD or Variance when every observation is multiplied by a constant 'k'.
    • Shortcut: The new SD is exactly |k| * (Old SD). The new Variance is exactly k² * (Old Variance).
    • Example: The variance of a dataset is 4. If every observation is multiplied by 3, find the new variance. Answer: 3² * 4 = 9 * 4 = 36.

    3. The Linear Equation Hack

    • Pattern: If y = ax + b, and the variance/SD of x is given, find the variance/SD of y.
    • Shortcut: Ignore 'b' completely. SD of y = |a| * (SD of x). Variance of y = a² * (Variance of x).
    • Example: If y = -2x + 5 and Variance of x is 3. Find Variance of y. Ignore 5. Answer: (-2)² * 3 = 4 * 3 = 12.

    GROUP 2: Standard Sequences (Instant Formulas)

    4. Variance of First 'n' Natural Numbers

    • Pattern: Find the variance or SD of the numbers 1, 2, 3, ... n.
    • Shortcut: The variance is ALWAYS exactly (n² - 1) / 12.
    • Example: Find the variance of the first 11 natural numbers. Here n=11. Answer: (11² - 1) / 12 = 120 / 12 = 10.

    5. Variance of an Arithmetic Progression (A.P.)

    • Pattern: Find the variance of a sequence of 'n' terms in an A.P. with a common difference 'd'.
    • Shortcut: The variance is ALWAYS exactly [d² * (n² - 1)] / 12.
    • Example: Find the variance of 3, 6, 9, 12, 15. Here n=5, d=3. Answer: [3² * (5² - 1)] / 12 = [9 * 24] / 12 = 18.

    6. Variance of First 'n' Even/Odd Numbers

    • Pattern: Find the variance of the first 'n' EVEN natural numbers or first 'n' ODD natural numbers.
    • Shortcut: Both have the exact same variance! It is ALWAYS exactly (n² - 1) / 3.
    • Example: Find the variance of the first 7 even natural numbers. Here n=7. Answer: (7² - 1) / 3 = 48 / 3 = 16.

    7. The "Two Number" SD Hack

    • Pattern: Find the Standard Deviation of exactly two numbers 'a' and 'b'.
    • Shortcut: Do not use the long variance formula. The SD is ALWAYS exactly |a - b| / 2 (half of their absolute difference).
    • Example: Find the SD of 14 and 20. Answer: |14 - 20| / 2 = 6 / 2 = 3.

    GROUP 3: Theoretical Minima & Maxima (No Calculations)

    8. The Absolute Deviation Minimum

    • Pattern: Find the value of 'A' for which the sum of absolute deviations Sum|xᵢ - A| is minimum.
    • Shortcut: The sum of absolute deviations is ALWAYS minimum when measured about the MEDIAN.
    • Example: (Matches pattern). Answer: A = Median.

    9. The Squared Deviation Minimum

    • Pattern: Find the value of 'A' for which the sum of squared deviations Sum(xᵢ - A)² is minimum.
    • Shortcut: The sum of squared deviations is ALWAYS minimum when measured about the MEAN.
    • Example: For a dataset, Sum(xᵢ - A)² is minimum. What is A? Answer: The Arithmetic Mean.

    10. The Maximum Variance Hack

    • Pattern: Find the maximum possible variance of a dataset where all values are strictly bounded in the interval [a, b].
    • Shortcut: The absolute maximum variance is ALWAYS exactly (b - a)² / 4.
    • Example: The marks of 50 students lie between 20 and 80. Find the maximum possible variance. Here a=20, b=80. Answer: (80 - 20)² / 4 = 3600 / 4 = 900.

    GROUP 4: Corrections & Combinations

    11. The Correct Mean Hack

    • Pattern: Correcting the mean when an incorrect observation was recorded.
    • Shortcut: Correct Mean = Old Mean + (Right Value - Wrong Value) / n.
    • Example: Mean of 20 observations is 50. One observation 30 was wrongly recorded as 80. Correct Mean = 50 + (30 - 80) / 20 = 50 - (50/20) = 50 - 2.5 = 47.5.

    12. Combined Variance (Same Mean)

    • Pattern: Find the combined variance of two groups that happen to have the EXACT SAME mean.
    • Shortcut: Do not use the massive d1, d2 formula. The combined variance is simply the weighted average: (n1V1 + n2V2) / (n1 + n2).
    • Example: Group A (n=10, V=4, Mean=50) and Group B (n=20, V=7, Mean=50). Answer: (104 + 207) / 30 = (40 + 140) / 30 = 180 / 30 = 6.

    13. The Consistency / Stability Checker

    • Pattern: A word problem asks which of two datasets (like batsmans' scores or factory wages) is more "consistent", "stable", or "uniform".
    • Shortcut: Find the Coefficient of Variation (CV) = (SD / Mean) * 100. The dataset with the LESSER CV is ALWAYS the more consistent/stable one.
    • Example: (Matches pattern). Answer: Calculate CV for both, pick the smaller one.

    GROUP 5: Standard Distribution Ratios

    14. The 4:5:6 Ratio Hack (Symmetric Distributions)

    • Pattern: Find the relationship between Quartile Deviation (QD), Mean Deviation (MD), and Standard Deviation (SD) for a normal or symmetric distribution.
    • Shortcut: Memorize the ratio 4:5:6. This means QD / 4 = MD / 5 = SD / 6.
    • Example: If SD = 12, find QD. QD / 4 = 12 / 6 -> QD / 4 = 2 -> QD = 8.

    15. The Empirical Formula Trick

    • Pattern: Find Mean, Median, or Mode when the other two are given.
    • Shortcut: Mode = 3*(Median) - 2*(Mean). (Memorize it as "Mode = 3 Med - 2 Mean").
    • Example: In a moderately skewed distribution, Mean = 20, Median = 22. Find Mode. Answer: Mode = 3(22) - 2(20) = 66 - 40 = 26.
